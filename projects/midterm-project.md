# 중간 실기 — Python 성적 분석기

## 목표

8주까지 배운 함수, 컬렉션, CSV, 예외 처리, 클래스 중 적절한 요소를 조합해 `datasets/scores.csv`를 분석합니다.

## 필수 기능

1. CSV를 읽고 점수를 숫자로 변환합니다.
2. 전체 평균, 최고 점수, 전공별 평균을 출력합니다.
3. 0~100 범위를 벗어난 점수를 탐지합니다.
4. 핵심 계산을 두 개 이상의 함수로 분리합니다.
5. 정상·빈 데이터·경계값 중 두 종류 이상을 테스트합니다.
6. AI Prompt와 직접 수정·검증 내용을 기록합니다.

## 제출물

- `midterm_<student_id>.ipynb`
- 1쪽 기술 메모: 설계, 오류, AI 활용, 한계
- 실행 결과

평가는 `rubrics/midterm-rubric.md`를 따릅니다.
