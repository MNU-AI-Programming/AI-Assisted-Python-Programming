## 코드 예제

```python
import pandas as pd
url="https://raw.githubusercontent.com/MNU-AI-Programming/AI-Assisted-Python-Programming/main/datasets/scores.csv"
df=pd.read_csv(url)
display(df.head())
print(df.shape)
print(df.columns.tolist())
print(df.dtypes)
```
