# AI 활용 코딩

## Chapter 8. AI 활용 프로젝트

비전공자를 위한 생성형 AI 활용 프로그래밍

---

# 학습 목표

- 프로젝트 주제를 정할 수 있다.
- AI에게 기획과 코드 생성을 요청할 수 있다.
- Colab에서 프로그램을 구현할 수 있다.
- AI가 만든 코드를 수정할 수 있다.
- 프로젝트 결과를 정리하고 발표할 수 있다.

---

# 프로젝트란?

지금까지 배운 내용을 활용하여 하나의 결과물을 만드는 활동입니다.

활용 내용:

- 입력
- 계산
- 조건문
- 반복문
- 함수
- 리스트
- 파일 저장
- AI Prompt

---

# 프로젝트의 핵심

어려운 프로그램을 만드는 것이 목표가 아닙니다.

중요한 것은:

- 문제를 설명하기
- AI에게 질문하기
- 코드를 실행하기
- 오류를 수정하기
- 직접 개선하기
- 과정을 기록하기

---

# 프로젝트 진행 흐름

```text
문제 찾기
↓
주제 정하기
↓
기능 목록 작성
↓
AI에게 기획 요청
↓
코드 생성 요청
↓
Colab 실행
↓
오류 수정
↓
결과 정리
```

---

# 프로젝트 주제 예시

| 프로젝트 | 설명 |
|---|---|
| 용돈 관리 | 수입과 지출 계산 |
| 여행 경비 | 총 여행 비용 계산 |
| 성적 관리 | 점수 평균 계산 |
| TODO 리스트 | 할 일 목록 관리 |
| 독서 기록 | 책 제목과 감상 저장 |

---

# 좋은 프로젝트 주제

너무 큰 주제는 피합니다.

나쁜 예:

```text
인공지능 비서 만들기
온라인 쇼핑몰 만들기
```

좋은 예:

```text
하루 용돈 기록 프로그램
여행 경비 계산기
학생 5명의 평균 계산기
```

---

# 프로젝트 기획서

```markdown
# 프로젝트 기획서

## 프로젝트 이름
## 만들고 싶은 이유
## 주요 사용자
## 주요 기능
## 입력값

## 사용할 Python 문법
## AI에게 도움받을 부분
```

---

# AI에게 기획 요청하기

```text
저는 프로그래밍을 처음 배우는 대학생입니다.

Google Colab에서 실행할 수 있는
용돈 관리 프로그램을 만들고 싶습니다.

목적, 주요 기능, 입력값, 출력값,
사용할 Python 문법을 표로 정리해 주세요.
```

---

# 기능은 3~5개

예: 용돈 관리 프로그램

1. 수입 입력
2. 지출 입력
3. 잔액 계산
4. 기록 출력
5. CSV 저장

작게 시작하는 것이 좋습니다.

---

# 코드 생성 Prompt

```text
Python으로 용돈 관리 프로그램을 만들어 주세요.

기능:
1. 수입 금액 입력
2. 지출 금액 입력
3. 잔액 계산
4. 결과 출력

조건:
- input(), int(), f-string 사용
- 초보자용 주석 포함
- Colab에서 실행 가능
```

---

# AI 코드 이해하기

AI가 만든 코드를 실행한 후 확인합니다.

- 어떤 입력을 받는가?
- 어떤 변수를 사용하는가?
- 조건문이 있는가?
- 반복문이 있는가?
- 함수가 있는가?
- 내가 수정할 부분은 어디인가?

---

# 디버깅 Prompt

```text
아래 Python 코드를 Colab에서 실행했더니 오류가 발생했습니다.

오류의 원인을 초보자도 이해할 수 있도록 설명하고,
수정된 코드를 제시해 주세요.

코드:
...

오류 메시지:
...
```

---

# 리팩토링 Prompt

```text
아래 Python 코드를 초보자도 이해하기 쉬운 형태로 리팩토링해 주세요.

기능은 그대로 유지하고,
반복되는 부분은 함수로 정리해 주세요.
```

---

# 예시 1. 용돈 관리

```python
income = int(input("이번 주 수입을 입력하세요: "))
expense = int(input("이번 주 지출을 입력하세요: "))

balance = income - expense

print(f"수입: {income}원")
print(f"지출: {expense}원")
print(f"잔액: {balance}원")
```

---

# 예시 2. TODO 리스트

```python
todo_list = []

for i in range(3):
    todo = input("할 일을 입력하세요: ")
    todo_list.append(todo)

for todo in todo_list:
    print(f"- {todo}")
```

---

# 예시 3. 성적 관리

```python
names = []
scores = []

student_count = int(input("학생 수: "))

for i in range(student_count):
    name = input("이름: ")
    score = int(input("점수: "))
    names.append(name)
    scores.append(score)

average = sum(scores) / len(scores)
print(f"평균: {average:.2f}")
```

# AI 활용 윤리

- AI가 만든 코드를 그대로 제출하지 않기
- 사용한 Prompt 기록하기
- 직접 수정한 부분 설명하기
- 코드 의미 이해하기
- 실행 결과 직접 확인하기
- 다른 사람 결과물 복사하지 않기

---

# 평가 기준

| 평가 요소 | 설명 |
|---|---|
| 문제 이해 | 목적이 분명한가 |
| 기능 구현 | 실행되는가 |
| AI 활용 | Prompt 기록 |
| 코드 이해 | 설명 가능한가 |
| 오류 해결 | 수정 과정 |
| 결과 정리 | 보고서 |
| 발표 | 설명 능력 |

---

# 핵심 정리

- 프로젝트는 배운 내용을 종합하는 활동입니다.
- 작고 쉬운 주제로 시작합니다.
- AI는 기획, 코드 생성, 디버깅에 활용합니다.
- AI가 만든 코드는 반드시 수정하고 이해합니다.
- 결과보다 과정 기록이 중요합니다.

---

# 마무리

AI 활용 코딩의 핵심은

> AI가 대신 만든 결과물이 아니라  
> AI와 함께 문제를 해결한 과정입니다.
