## 8.1 프로젝트란?

프로젝트는 지금까지 배운 내용을 활용하여 하나의 결과물을 만드는 활동입니다.

- 변수

- `input()`
- 계산
- 조건문
- 반복문
- 함수
- 리스트
- 파일 저장
- AI Prompt
- 디버깅
- 리팩토링

## 8.2 프로젝트의 핵심 목표

이번 프로젝트의 목표는 어려운 프로그램을 만드는 것이 아닙니다.

비전공자 수준에서 중요한 것은 다음입니다.

1. 만들고 싶은 프로그램을 설명할 수 있다.
2. AI에게 적절한 Prompt를 작성할 수 있다.
3. AI가 만든 코드를 Google Colab에서 실행할 수 있다.
4. 오류가 나면 수정할 수 있다.
5. 자신에게 맞게 프로그램을 바꿀 수 있다.
6. 프로젝트 과정을 글로 정리할 수 있다.

## 8.5 프로젝트 주제 선정 방법

좋은 프로젝트 주제는 너무 크지 않아야 합니다.

나쁜 예:

```text
인공지능 비서 만들기
온라인 쇼핑몰 만들기
대학 전체 수강신청 시스템 만들기
```

좋은 예:

```text
하루 용돈 기록 프로그램
여행 경비 합계 계산기
책 3권의 독서 기록 저장 프로그램
학생 5명의 점수 평균 계산기
```

처음에는 작게 시작하는 것이 좋습니다.

작은 프로그램을 완성한 뒤 기능을 하나씩 추가하면 됩니다.

## 8.10 디버깅하기

프로젝트를 하다 보면 오류가 발생할 수 있습니다.

오류가 발생하면 다음 순서로 해결합니다.

1. 오류 메시지를 읽는다.
2. 어느 줄에서 오류가 났는지 확인한다.
3. 오류가 난 코드와 오류 메시지를 AI에게 보여 준다.
4. AI의 설명을 읽는다.
5. 수정한 코드를 다시 실행한다.

## 8.11 리팩토링하기

프로그램이 실행된다고 끝이 아닙니다.

코드를 더 읽기 좋게 바꾸는 과정도 중요합니다.

리팩토링할 때 AI에게 다음과 같이 요청할 수 있습니다.

```text
아래 Python 코드를 초보자도 이해하기 쉬운 형태로 리팩토링해 주세요.

기능은 그대로 유지해 주세요.

반복되는 부분은 함수로 정리해 주세요.

변수 이름은 의미 있게 바꿔 주세요.

각 함수에 간단한 주석을 달아 주세요.

코드:
여기에 코드 붙여 넣기
```

## 8.12 프로젝트 예시 1: 용돈 관리 프로그램

아래는 기본적인 용돈 관리 프로그램입니다.

```python
income = int(input("이번 주 수입을 입력하세요: "))
expense = int(input("이번 주 지출을 입력하세요: "))

balance = income - expense

print("===== 용돈 관리 결과 =====")
print(f"수입: {income}원")
print(f"지출: {expense}원")
print(f"잔액: {balance}원")
```

---

### 기능 개선 아이디어

위 프로그램에 다음 기능을 추가할 수 있습니다.

- 지출 항목 입력하기
- 여러 번 지출 입력받기
- 총 지출 계산하기
- CSV 파일로 저장하기
- 잔액이 0보다 작으면 경고 메시지 출력하기

## 8.13 프로젝트 예시 2: TODO 리스트

할 일을 리스트에 저장하는 프로그램입니다.

```python
todo_list = []

for i in range(3):
    todo = input("할 일을 입력하세요: ")
    todo_list.append(todo)

print("===== TODO 리스트 =====")

for todo in todo_list:
    print(f"- {todo}")
```

---

### 기능 개선 아이디어

- 할 일 개수를 사용자가 정하기
- 완료 여부 표시하기
- 파일로 저장하기
- AI에게 더 보기 좋은 출력 형식 요청하기

## 8.14 프로젝트 예시 3: 성적 관리 프로그램

Chapter 7에서 배운 내용을 확장할 수 있습니다.

```python
names = []
scores = []

student_count = int(input("학생 수를 입력하세요: "))

for i in range(student_count):
    name = input("학생 이름을 입력하세요: ")
    score = int(input("점수를 입력하세요: "))

    names.append(name)
    scores.append(score)

average = sum(scores) / len(scores)

print("===== 성적 결과 =====")

for i in range(len(names)):
    print(f"{names[i]}: {scores[i]}점")

print(f"평균 점수: {average:.2f}")
```

---

### 기능 개선 아이디어

- 최고 점수 출력
- 최저 점수 출력
- 학점 출력
- CSV 파일 저장
- 평균 이상 학생만 출력

## 8.17 AI 활용 윤리

프로젝트에서 AI를 사용하는 것은 허용됩니다.

하지만 다음을 반드시 지켜야 합니다.

- AI가 만든 코드를 그대로 제출하지 않는다.
- 사용한 Prompt를 기록한다.
- 직접 수정한 부분을 설명한다.
- 코드의 의미를 이해하려고 노력한다.
- 실행 결과를 직접 확인한다.
- 다른 사람의 결과물을 복사하지 않는다.

## 8.18 프로젝트 평가 기준

프로젝트는 단순히 코드가 길다고 좋은 평가를 받는 것이 아닙니다.

중요한 것은 다음입니다.

| 평가 요소 | 설명 |
|---|---|
| 문제 이해 | 무엇을 해결하려는지 분명한가? |
| 기능 구현 | 기본 기능이 실행되는가? |
| AI 활용 과정 | Prompt와 수정 과정이 기록되어 있는가? |
| 코드 이해 | 코드의 의미를 설명할 수 있는가? |
| 오류 해결 | 오류를 수정한 과정이 있는가? |
| 결과 정리 | 실행 결과와 느낀 점이 정리되어 있는가? |
| 발표 | 자신이 만든 프로그램을 설명할 수 있는가? |
