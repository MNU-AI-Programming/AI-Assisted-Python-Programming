# Markdown 기초

> Markdown은 GitHub에서 문서를 작성할 때 많이 사용하는 간단한 문서 작성 문법입니다.

---

## 1. 제목

```markdown
# 제목 1
## 제목 2
### 제목 3
```

결과:

# 제목 1
## 제목 2
### 제목 3

---

## 2. 목록

```markdown
- 항목 1
- 항목 2
- 항목 3
```

번호 목록:

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
```

---

## 3. 굵게와 기울임

```markdown
**굵게**
*기울임*
```

---

## 4. 코드 블록

Python 코드를 작성할 때는 다음과 같이 씁니다.

````markdown
```python
print("Hello")
```
````

---

## 5. 표

```markdown
| 이름 | 점수 |
|---|---:|
| 김민수 | 90 |
| 이서연 | 85 |
```

결과:

| 이름 | 점수 |
|---|---:|
| 김민수 | 90 |
| 이서연 | 85 |

---

## 6. 링크

```markdown
[Google Colab](https://colab.research.google.com/)
```

---

## 7. 이미지

```markdown
![이미지 설명](images/example.png)
```

---

## 8. 과제 보고서 기본 양식

```markdown
# 과제 제목

## 1. 사용한 AI Prompt

## 2. 최종 코드

## 3. 실행 결과

## 4. 직접 수정한 부분

## 5. 느낀 점
```

---

## 9. 프로젝트 보고서 기본 양식

```markdown
# 프로젝트 결과 보고서

## 1. 프로젝트 제목

## 2. 프로젝트 목적

## 3. 주요 기능

## 4. 사용한 Python 문법

## 5. 사용한 AI Prompt

## 6. 최종 코드

## 7. 실행 결과

## 8. 오류와 해결 과정

## 9. 직접 수정한 부분

## 10. 느낀 점
```
