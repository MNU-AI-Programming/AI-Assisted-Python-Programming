# Google Colab 사용법

> 이 교재의 모든 실습은 Google Colab에서 진행합니다.  
> Google Colab은 웹 브라우저에서 Python 코드를 실행할 수 있는 무료 실습 환경입니다.

---

## 1. Google Colab이란?

Google Colab은 설치 없이 Python 코드를 실행할 수 있는 온라인 노트북 환경입니다.

장점은 다음과 같습니다.

- Python 설치가 필요 없음
- VS Code 설치가 필요 없음
- 웹 브라우저만 있으면 실행 가능
- Google Drive에 저장 가능
- 링크 공유 가능
- 수업 실습에 적합

---

## 2. Colab 노트북 열기

GitHub에서 `practice.ipynb` 파일을 클릭하면 노트북 내용을 볼 수 있습니다.

Colab에서 열려면 다음 방법 중 하나를 사용합니다.

### 방법 1. Open in Colab 버튼 사용

각 Chapter README에 있는 **Open in Colab** 버튼을 클릭합니다.

### 방법 2. Colab에서 GitHub 파일 열기

1. Google Colab 접속
2. `파일 > 노트 열기`
3. `GitHub` 탭 선택
4. 저장소 주소 입력
5. 원하는 `practice.ipynb` 선택

---

## 3. 코드 셀 실행하기

Colab에는 두 종류의 셀이 있습니다.

| 셀 종류 | 설명 |
|---|---|
| 텍스트 셀 | 설명, 제목, 메모 작성 |
| 코드 셀 | Python 코드 작성 및 실행 |

코드 셀 왼쪽의 실행 버튼을 누르면 코드가 실행됩니다.

단축키:

```text
Shift + Enter
```

---

## 4. 실습 파일 저장하기

Colab에서 실습한 내용은 Google Drive에 저장할 수 있습니다.

1. `파일`
2. `드라이브에 사본 저장`
3. 자신의 Google Drive에 저장
4. 파일 이름을 과제명에 맞게 수정

예:

```text
chapter03_assignment_홍길동.ipynb
```

---

## 5. 과제 링크 공유하기

과제 제출 시 Colab 링크를 공유해야 할 수 있습니다.

1. Colab 오른쪽 위 `공유` 클릭
2. 링크 보기 권한 설정
3. 링크 복사
4. LMS 또는 GitHub에 제출

권장 설정:

```text
링크가 있는 모든 사용자: 보기 가능
```

---

## 6. 코드가 실행되지 않을 때

다음을 확인하세요.

- 코드 셀을 실행했는가?
- 위쪽 셀부터 순서대로 실행했는가?
- 변수 이름을 잘못 입력하지 않았는가?
- 따옴표와 괄호가 빠지지 않았는가?
- 들여쓰기가 맞는가?
- 오류 메시지를 읽었는가?

오류가 발생하면 오류 메시지를 복사하여 ChatGPT에게 질문할 수 있습니다.

---

## 7. 오류 해결 Prompt

```text
아래 Python 코드를 Google Colab에서 실행했더니 오류가 발생했습니다.

오류 메시지를 초보자도 이해할 수 있도록 설명하고,
수정된 코드를 제시해 주세요.

코드:
[여기에 코드 붙여 넣기]

오류 메시지:
[여기에 오류 메시지 붙여 넣기]
```

---

## 8. 제출 전 확인 사항

과제를 제출하기 전에 다음을 확인하세요.

- 모든 코드 셀이 실행되는가?
- 실행 결과가 보이는가?
- 사용한 AI Prompt를 기록했는가?
- 직접 수정한 부분을 설명했는가?
- Colab 링크 권한이 열려 있는가?

---

## 9. Colab 사용 예절

- 다른 학생의 Colab 파일을 복사하지 않습니다.
- AI가 만든 코드를 그대로 제출하지 않습니다.
- 실행 결과를 직접 확인합니다.
- 오류가 발생한 과정을 기록합니다.
