# Chapter 5 과제
## AI와 함께 반복문 프로그램 만들기

---

## 과제 목표

이번 과제의 목표는 `for`, `while`, `range()`, 조건문을 활용하여 반복 실행되는 프로그램을 만드는 것입니다.

---

## 과제 설명

ChatGPT를 활용하여 **반복문 프로그램**을 작성하세요.

아래 주제 중 하나를 선택합니다.

1. 구구단 출력 프로그램
2. 숫자 맞추기 게임
3. 1부터 N까지 합계 계산기
4. 짝수/홀수 출력 프로그램
5. 사용자가 0을 입력할 때까지 숫자를 입력받는 프로그램
6. 자유 주제 반복문 프로그램

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. `for` 또는 `while` 중 하나 이상 사용할 것
3. `input()` 함수를 1회 이상 사용할 것
4. 조건문 `if`를 1회 이상 사용할 것
5. f-string을 1회 이상 사용할 것
6. ChatGPT에 사용한 Prompt를 기록할 것
7. AI가 만든 코드를 자신의 주제에 맞게 수정할 것
8. 최소 2가지 입력값으로 테스트한 결과를 기록할 것

---

## 추천 Prompt 1: 구구단 프로그램

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Google Colab에서 실행할 수 있는 구구단 프로그램을 만들고 싶습니다.

사용자가 출력할 단을 입력하면 해당 단의 구구단을 1부터 9까지 출력하게 해 주세요.

for, range(), input(), int(), f-string을 사용해 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

---

## 추천 Prompt 2: 숫자 맞추기 게임

```text
Python으로 숫자 맞추기 게임을 만들고 싶습니다.

1부터 10 사이의 정답 숫자를 정하고,
사용자가 정답을 맞힐 때까지 계속 숫자를 입력받게 해 주세요.

while, if, elif, else, input(), int()를 사용해 주세요.

정답보다 작으면 "더 큰 숫자입니다", 크면 "더 작은 숫자입니다"를 출력하게 해 주세요.

프로그래밍 초보자도 이해할 수 있도록 주석을 달아 주세요.
```

---

## 추천 Prompt 3: 합계 계산기

```text
Python으로 1부터 사용자가 입력한 숫자 N까지의 합계를 구하는 프로그램을 만들어 주세요.

input(), int(), for, range(), f-string을 사용해 주세요.

각 줄에 초보자용 주석을 달아 주세요.
```

## 주의사항

`while`을 사용할 경우 무한 반복이 발생하지 않도록 반드시 종료 조건을 확인해야 합니다.  
AI가 만든 반복문 코드는 `range()` 범위와 종료 조건을 직접 검토해야 합니다.
