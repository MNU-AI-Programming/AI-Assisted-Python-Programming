## 3. 최종 코드

```python
def calculate_total(price, quantity):
    return price * quantity

price = int(input("상품 가격을 입력하세요: "))
quantity = int(input("수량을 입력하세요: "))

total = calculate_total(price, quantity)

print(f"총 금액은 {total}원입니다.")
```
