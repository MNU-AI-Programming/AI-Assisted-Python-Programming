# 리팩토링 Prompt

> 리팩토링은 코드의 기능은 유지하면서 더 읽기 좋고 관리하기 쉽게 개선하는 과정입니다.

---

## 1. 초보자용 리팩토링

```text
아래 Python 코드를 초보자도 이해하기 쉬운 형태로 리팩토링해 주세요.

기능은 그대로 유지해 주세요.

변수 이름을 의미 있게 바꾸고,
각 줄에 필요한 주석을 달아 주세요.

코드:
[여기에 코드 붙여 넣기]
```
