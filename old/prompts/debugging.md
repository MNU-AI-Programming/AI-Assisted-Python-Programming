# 디버깅 Prompt

> 디버깅은 오류를 찾고 수정하는 과정입니다.  
> AI에게 오류 해결을 요청할 때는 코드와 오류 메시지를 함께 제공해야 합니다.

---

## 1. 기본 오류 해결 Prompt

```text
아래 Python 코드를 Google Colab에서 실행했더니 오류가 발생했습니다.

오류의 원인을 초보자도 이해할 수 있도록 설명해 주세요.

수정된 코드도 함께 제시해 주세요.

코드:
[여기에 코드 붙여 넣기]

오류 메시지:
[여기에 오류 메시지 붙여 넣기]
```
