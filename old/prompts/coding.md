# 코드 생성 Prompt

---

## 1. Python 개념 설명 요청

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 비전공 대학생입니다.

[개념 이름]을 쉽게 설명해 주세요.

일상생활 비유를 사용해 설명하고,
Google Colab에서 실행할 수 있는 예제 코드 3개를 만들어 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

---

## 2. 간단한 프로그램 생성 요청

```text
Python으로 [프로그램 이름]을 만들고 싶습니다.

기능:
1. [기능 1]
2. [기능 2]
3. [기능 3]

조건:
- Google Colab에서 실행 가능해야 합니다.
- input()을 사용해 주세요.
- f-string을 사용해 주세요.
- 초보자도 이해할 수 있도록 주석을 달아 주세요.
- 너무 어려운 문법은 사용하지 마세요.
```

---

## 3. 조건문 프로그램 요청

```text
Python으로 [조건 판단 프로그램]을 만들어 주세요.

입력값:
- [입력값 설명]

조건:
- [조건 1]
- [조건 2]
- [조건 3]

출력:
- [출력 내용]

if, elif, else를 사용해 주세요.
초보자용 주석을 달아 주세요.
```

---

## 4. 반복문 프로그램 요청

```text
Python으로 [반복 프로그램]을 만들어 주세요.

반복 조건:
- [몇 번 반복하는지 또는 언제까지 반복하는지]

사용 문법:
- for 또는 while
- input()
- f-string

Google Colab에서 실행할 수 있게 작성하고,
무한 반복이 발생하지 않도록 해 주세요.
```
