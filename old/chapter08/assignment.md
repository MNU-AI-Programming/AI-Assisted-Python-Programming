# Chapter 8 최종 프로젝트
## AI와 함께 나만의 Python 프로그램 만들기

---

## 프로젝트 목표

이번 최종 프로젝트의 목표는 지금까지 배운 Python 기초 문법과 생성형 AI 활용 방법을 바탕으로 하나의 프로그램을 완성하는 것입니다.

---

## 프로젝트 주제

아래 주제 중 하나를 선택하거나 자유 주제를 정할 수 있습니다.

1. 용돈 관리 프로그램
2. 여행 경비 계산기
3. 성적 관리 프로그램
4. TODO 리스트
5. 독서 기록 프로그램
6. 카페 주문 계산기
7. 운동 기록 프로그램
8. 자유 주제 프로그램

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. `input()` 함수를 사용할 것
3. 변수와 f-string을 사용할 것
4. 조건문 또는 반복문 중 하나 이상 사용할 것
5. 함수 또는 리스트 중 하나 이상 사용할 것
6. ChatGPT에 사용한 Prompt를 2개 이상 기록할 것
7. AI가 만든 코드를 직접 수정한 부분을 설명할 것
8. 실행 결과를 포함할 것
9. 오류가 있었다면 해결 과정을 기록할 것
10. 프로젝트 결과 보고서를 작성할 것

---

## 선택 조건

아래 항목은 필수는 아니지만 포함하면 더 좋은 평가를 받을 수 있습니다.

- CSV 파일 저장
- 여러 명의 데이터 처리
- 함수 2개 이상 사용
- 입력값 검증
- 메뉴 반복 기능
- 결과를 표처럼 출력
- 프로젝트 발표 자료 작성

---

## 프로젝트 진행 단계

### 1단계. 주제 정하기

다음 질문에 답합니다.

```markdown
내가 만들 프로그램은 무엇인가?
왜 이 프로그램을 만들고 싶은가?
누가 사용할 수 있는가?
```

---

### 2단계. 기능 정하기

기능은 3~5개 정도로 정합니다.

예:

```markdown
프로젝트명: 용돈 관리 프로그램

기능:
1. 수입 입력
2. 지출 입력
3. 잔액 계산
4. 지출 목록 출력
5. CSV 저장
```

---

### 3단계. AI에게 기획 요청하기

추천 Prompt:

```text
당신은 Python 프로젝트를 도와주는 대학 강사입니다.

저는 프로그래밍을 처음 배우는 비전공 대학생입니다.

Google Colab에서 실행할 수 있는 간단한 Python 프로젝트를 만들고 싶습니다.

주제는 "[내 주제]"입니다.

이 프로젝트의 목적, 주요 기능, 입력값, 출력값, 사용할 Python 문법을 표로 정리해 주세요.

너무 어렵지 않게 초보자 수준으로 제안해 주세요.
```

---

### 4단계. AI에게 코드 생성 요청하기

추천 Prompt:

```text
Python으로 Google Colab에서 실행할 수 있는 "[내 프로젝트명]" 프로그램을 만들어 주세요.

기능:
1. [기능 1]
2. [기능 2]
3. [기능 3]

조건:
- input()을 사용해 주세요.
- f-string을 사용해 주세요.
- 조건문 또는 반복문을 포함해 주세요.
- 함수 또는 리스트를 포함해 주세요.
- 초보자도 이해할 수 있도록 주석을 달아 주세요.
- 너무 어려운 문법은 사용하지 마세요.
```

---

### 5단계. 실행하고 수정하기

AI가 만든 코드를 Google Colab에 붙여 넣고 실행합니다.

확인할 점:

- 오류 없이 실행되는가?
- 입력값이 제대로 처리되는가?
- 출력 결과가 이해하기 쉬운가?
- 내 주제에 맞게 수정했는가?
- 코드의 의미를 설명할 수 있는가?

---

### 6단계. 디버깅하기

오류가 발생하면 아래 Prompt를 사용합니다.

```text
아래 Python 코드를 Google Colab에서 실행했더니 오류가 발생했습니다.

오류의 원인을 초보자도 이해할 수 있도록 설명하고,
수정된 코드를 제시해 주세요.

코드:
[내 코드]

오류 메시지:
[오류 메시지]
```

---

### 7단계. 결과 보고서 작성하기

다음 형식으로 결과 보고서를 작성합니다.

```markdown
# 최종 프로젝트 결과 보고서

## 1. 프로젝트 제목

## 2. 프로젝트를 만든 이유

## 3. 주요 기능

## 4. 사용한 Python 문법

## 5. 사용한 AI Prompt

### Prompt 1

### Prompt 2

## 6. 최종 코드

## 7. 실행 결과

## 8. 오류와 해결 과정

## 9. 직접 수정한 부분

## 10. 느낀 점
```

다음에 해당하면 감점될 수 있습니다.

## 프로젝트 예시 Prompt 모음

### 용돈 관리 프로그램

```text
Python으로 용돈 관리 프로그램을 만들어 주세요.

수입과 지출을 입력받아 잔액을 계산하고,
잔액이 0보다 작으면 경고 메시지를 출력하게 해 주세요.

input(), int(), if, f-string을 사용해 주세요.
```

### TODO 리스트

```text
Python으로 TODO 리스트 프로그램을 만들어 주세요.

사용자가 할 일 5개를 입력하면 리스트에 저장하고,
마지막에 전체 할 일 목록을 출력하게 해 주세요.

list, append(), for, input(), f-string을 사용해 주세요.
```

### 성적 관리 프로그램

```text
Python으로 성적 관리 프로그램을 만들어 주세요.

학생 이름과 점수를 입력받아 리스트에 저장하고,
평균 점수와 최고 점수를 출력하게 해 주세요.

list, append(), for, sum(), len(), max(), f-string을 사용해 주세요.
```

## 마지막 주의사항

최종 프로젝트에서 가장 중요한 것은 완벽한 코드가 아닙니다.

중요한 것은 다음입니다.

- AI에게 어떻게 질문했는가?
- AI가 만든 코드를 어떻게 이해했는가?
- 어떤 오류를 만났는가?
- 어떻게 수정했는가?
- 나만의 기능을 어떻게 추가했는가?

프로젝트 과정 자체가 학습 결과입니다.
