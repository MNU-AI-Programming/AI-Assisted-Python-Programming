## 서술형 문제

### 14. AI가 만든 코드를 그대로 제출하면 안 되는 이유를 설명하시오.

**예시 답안:**  
AI가 만든 코드는 틀릴 수 있고, 내가 이해하지 못한 코드일 수 있다.  
학습의 목적은 AI가 만든 코드를 이해하고 자신의 목적에 맞게 수정하는 것이다.  
따라서 사용한 Prompt와 수정한 부분을 기록하고, 실행 결과를 직접 확인해야 한다.

프로젝트 제목, 프로젝트 목적, 주요 기능, 사용한 Python 문법, 사용한 AI Prompt, 최종 코드, 실행 결과, 오류와 해결 과정, 직접 수정한 부분, 느낀 점 등을 포함해야 한다.
