# Chapter 4 과제
## AI와 함께 조건 판단 프로그램 만들기

---

## 과제 목표

이번 과제의 목표는 `if`, `elif`, `else`를 사용하여 조건에 따라 다른 결과를 출력하는 프로그램을 만드는 것입니다.

---

## 과제 설명

ChatGPT를 활용하여 **조건 판단 프로그램**을 작성하세요.

아래 주제 중 하나를 선택합니다.

1. 학점 계산기
2. 성인/미성년자 판별기
3. 로그인 프로그램
4. 날씨 메시지 프로그램
5. 할인율 계산 프로그램
6. 자유 주제 조건문 프로그램

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. `input()` 함수를 1회 이상 사용할 것
3. `if` 문을 사용할 것
4. `elif` 또는 `else`를 사용할 것
5. 비교 연산자를 사용할 것
6. f-string을 1회 이상 사용할 것
7. ChatGPT에 사용한 Prompt를 기록할 것
8. AI가 만든 코드를 자신의 주제에 맞게 수정할 것
9. 최소 3가지 입력값으로 테스트한 결과를 기록할 것

---

## 추천 Prompt 1: 학점 계산기

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Google Colab에서 실행할 수 있는 학점 계산기 프로그램을 만들고 싶습니다.

점수를 입력받아 다음 기준으로 학점을 출력하게 해 주세요.

90점 이상: A
80점 이상: B
70점 이상: C
60점 이상: D
60점 미만: F

input(), int(), if, elif, else, f-string을 사용해 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

---

## 추천 Prompt 2: 로그인 프로그램

```text
Python으로 간단한 로그인 프로그램을 만들고 싶습니다.

아이디와 비밀번호를 입력받고,
아이디가 admin이고 비밀번호가 1234이면 로그인 성공,
그렇지 않으면 로그인 실패를 출력하게 해 주세요.

if, else, and 연산자를 사용해 주세요.

프로그래밍 초보자도 이해할 수 있도록 주석을 달아 주세요.
```

---

## 추천 Prompt 3: 날씨 메시지 프로그램

```text
Python으로 온도에 따라 날씨 메시지를 출력하는 프로그램을 만들어 주세요.

온도를 입력받고,
30도 이상이면 "덥습니다",
20도 이상이면 "날씨가 좋습니다",
10도 이상이면 "쌀쌀합니다",
그 외에는 "춥습니다"를 출력하게 해 주세요.

if, elif, else를 사용하고 초보자용 주석을 달아 주세요.
```

## 3. 최종 코드

```python
score = int(input("점수를 입력하세요: "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"{score}점의 학점은 {grade}입니다.")
```

## 4. 테스트 결과

- 입력: 75 / 결과: C
- 입력: 55 / 결과: F
