## 4.1 조건문이란?

조건문은 조건에 따라 실행할 코드를 선택하는 문법입니다.

일상생활에서도 우리는 조건에 따라 행동합니다.

예를 들어 다음과 같습니다.

```text
비가 오면 우산을 가져간다.
배가 고프면 밥을 먹는다.
점수가 60점 이상이면 합격이다.
비밀번호가 맞으면 로그인한다.
```

Python에서도 이와 같은 판단을 할 수 있습니다.

---

## 4.2 if 문

가장 기본적인 조건문은 `if`입니다.

```python
score = 80

if score >= 60:
    print("합격입니다.")
```

위 코드는 `score`가 60 이상이면 `"합격입니다."`를 출력합니다.

---

### if 문의 구조

```python
if 조건:
    실행할 코드
```

주의할 점은 다음과 같습니다.

- 조건 뒤에는 콜론 `:`을 붙입니다.
- 실행할 코드는 들여쓰기 합니다.
- 보통 스페이스 4칸을 사용합니다.

---

## ⚠️ 자주 하는 실수 1: 콜론 빠뜨리기

잘못된 코드:

```python
if score >= 60
    print("합격입니다.")
```

올바른 코드:

```python
if score >= 60:
    print("합격입니다.")
```

조건문 줄 끝에는 반드시 콜론 `:`이 필요합니다.

---

## ⚠️ 자주 하는 실수 2: 들여쓰기 하지 않기

잘못된 코드:

```python
if score >= 60:
print("합격입니다.")
```

올바른 코드:

```python
if score >= 60:
    print("합격입니다.")
```

조건문 안에서 실행할 코드는 들여쓰기 해야 합니다.

## 4.3 비교 연산자

조건문에서는 값을 비교하기 위해 비교 연산자를 사용합니다.

| 연산자 | 의미 | 예시 |
|---|---|---|
| `>` | 크다 | `score > 60` |
| `<` | 작다 | `score < 60` |
| `>=` | 크거나 같다 | `score >= 60` |
| `<=` | 작거나 같다 | `score <= 60` |
| `==` | 같다 | `password == "1234"` |
| `!=` | 같지 않다 | `password != "1234"` |

---

### 비교 연산자 예제

```python
score = 75

print(score >= 60)
print(score < 60)
print(score == 75)
print(score != 75)
```

실행 결과:

```text
True
False
True
False
```

비교 연산의 결과는 `True` 또는 `False`입니다.

## 4.4 else 문

`else`는 조건이 거짓일 때 실행됩니다.

```python
score = 50

if score >= 60:
    print("합격입니다.")
else:
    print("불합격입니다.")
```

위 코드는 점수가 60 이상이면 합격, 그렇지 않으면 불합격을 출력합니다.

---

### if-else 구조

```python
if 조건:
    조건이 참일 때 실행
else:
    조건이 거짓일 때 실행
```

---

## 4.5 elif 문

조건이 여러 개일 때는 `elif`를 사용합니다.

`elif`는 “그렇지 않고, 만약 ~라면”이라는 뜻입니다.

예를 들어 점수에 따라 학점을 출력하려면 여러 조건이 필요합니다.

```python
score = 85

if score >= 90:
    print("A학점")
elif score >= 80:
    print("B학점")
elif score >= 70:
    print("C학점")
elif score >= 60:
    print("D학점")
else:
    print("F학점")
```

---

### 조건문의 순서가 중요합니다

학점 계산에서는 조건의 순서가 매우 중요합니다.

아래 코드는 잘못된 예입니다.

```python
score = 95

if score >= 60:
    print("D학점")
elif score >= 70:
    print("C학점")
elif score >= 80:
    print("B학점")
elif score >= 90:
    print("A학점")
```

95점은 60점 이상이므로 첫 번째 조건에서 바로 `"D학점"`이 출력됩니다.

따라서 높은 점수 조건부터 작성해야 합니다.

## 💡 AI Tip

AI가 만든 조건문 코드를 볼 때는 **조건의 순서**를 반드시 확인해야 합니다.

특히 학점, 등급, 할인율처럼 범위가 있는 문제는 조건 순서가 중요합니다.

## 4.6 논리 연산자

여러 조건을 함께 사용할 때는 논리 연산자를 사용합니다.

| 연산자 | 의미 | 예시 |
|---|---|---|
| `and` | 두 조건이 모두 참 | `age >= 20 and has_ticket == True` |
| `or` | 둘 중 하나라도 참 | `menu == "커피" or menu == "차"` |
| `not` | 참과 거짓을 반대로 | `not is_logged_in` |

---

### and 예제

```python
age = 20
has_ticket = True

if age >= 19 and has_ticket == True:
    print("입장 가능합니다.")
else:
    print("입장할 수 없습니다.")
```

나이가 19세 이상이고 티켓이 있어야 입장할 수 있습니다.

---

### or 예제

```python
menu = "커피"

if menu == "커피" or menu == "차":
    print("음료를 주문했습니다.")
else:
    print("음료가 아닙니다.")
```

## 4.7 학점 계산기 만들기

점수를 입력받아 학점을 출력하는 프로그램을 만들어 봅시다.

```python
score = int(input("점수를 입력하세요: "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"점수는 {score}점이고, 학점은 {grade}입니다.")
```

3. 90점, 80점, 70점, 60점 같은 경계값이 잘 처리되는가?
4. 출력 결과가 이해하기 쉬운가?

## 4.8 로그인 프로그램 만들기

이번에는 아이디와 비밀번호를 확인하는 프로그램을 만들어 봅시다.

```python
user_id = input("아이디를 입력하세요: ")
password = input("비밀번호를 입력하세요: ")

if user_id == "admin" and password == "1234":
    print("로그인 성공")
else:
    print("로그인 실패")
```

이 프로그램은 아이디가 `"admin"`이고 비밀번호가 `"1234"`일 때만 로그인 성공을 출력합니다.

---

### 조건 자세히 보기

```python
user_id == "admin" and password == "1234"
```

이 조건은 두 가지가 모두 참이어야 합니다.

1. `user_id`가 `"admin"`인가?
2. `password`가 `"1234"`인가?

둘 중 하나라도 틀리면 로그인 실패입니다.

---

## ⚠️ 자주 하는 실수 3: `=`와 `==` 혼동

Python에서 `=`와 `==`는 다릅니다.

| 기호 | 의미 |
|---|---|
| `=` | 값을 저장 |
| `==` | 같은지 비교 |

예:

```python
password = "1234"
```

이 코드는 `"1234"`를 `password` 변수에 저장합니다.

```python
password == "1234"
```

이 코드는 `password`가 `"1234"`와 같은지 비교합니다.
