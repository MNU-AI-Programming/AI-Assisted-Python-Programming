# Week 7. Class와 객체지향 프로그래밍 기초 영상 대본
## 예상 길이: 50~55분

### [00:00~05:00] 도입과 이전 주 복습

안녕하세요. 국립목포대학교 컴퓨터학부 이영호입니다. 이번 시간은 Week 7, **Class와 객체지향 프로그래밍 기초**입니다. 오늘도 코드 복사보다 문제 분석, 실행, 테스트, 수정 과정을 중심으로 학습합니다.

### [05:00~10:00] 문제 상황과 학습 목표

오늘 해결할 문제는 다음과 같습니다.

> Student class에 이름·전공·점수와 학점 판정·요약 메서드를 구현합니다.

학생에게 입력과 예상 결과를 먼저 적도록 안내합니다.

### [10:00~20:00] 핵심 개념

- **class**: 객체의 데이터와 행동을 정의하는 설계도입니다.
- **객체**: class를 바탕으로 생성된 실제 데이터 단위입니다.
- **속성**: 객체가 보유하는 데이터입니다.
- **메서드**: 객체가 수행하는 행동입니다.

각 개념이 오늘 문제의 어느 부분에 사용되는지 연결합니다.

### [20:00~30:00] Colab 기본 예제

GitHub Week 7 폴더에서 `practice.ipynb`를 열고 Open in Colab 버튼을 클릭합니다. 실행 전에 결과를 예측하고, 실행 후 변수·함수·데이터 열을 확인합니다.

### [30:00~38:00] AI Prompt와 답변 검토

문제를 입력·처리·출력으로 정리하고 코드와 테스트를 작성해 달라는 Prompt를 입력합니다. AI 답변을 기존 코드와 비교하고 수용·수정·미사용 부분을 표시합니다.

### [38:00~45:00] TODO와 디버깅

TODO 셀은 학생이 직접 작성합니다. 오류 셀에서는 오류 유형, 마지막 메시지, 발생 줄을 확인하고 AI에게 코드·오류·예상 결과를 함께 제공합니다.

### [45:00~50:00] 테스트

정상 입력, 경계값, 예외 입력을 실행합니다. assert 결과를 확인하고 학생이 테스트 하나를 추가합니다.

### [50:00~55:00] 정리와 과제

오늘의 핵심은 AI 코드 생성이 아니라 검증·수정·설명입니다. 과제는 **Student class와 객체 리스트 리팩토링**이며 Prompt, 테스트, 수정 내역을 포함합니다.
