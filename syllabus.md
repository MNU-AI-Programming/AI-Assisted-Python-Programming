## 2. 교과목 설명

본 교과목은 프로그래밍을 처음 배우는 비전공 대학생을 대상으로 생성형 AI를 활용하여 Python 프로그래밍의 기초를 학습하는 교양 실습 과목이다.

학생들은 Python 문법을 단순히 암기하는 것이 아니라, ChatGPT와 같은 생성형 AI에게 문제를 설명하고, AI가 생성한 코드를 이해·실행·수정하는 과정을 통해 AI 시대의 기초 코딩 역량을 기른다.

모든 실습은 Google Colab에서 진행되며, 별도의 프로그램 설치 없이 웹 브라우저만으로 Python 코드를 작성하고 실행한다. 최종적으로 학생들은 AI와 협업하여 간단한 Python 프로젝트를 완성하고, 사용한 Prompt와 수정 과정을 정리한다.

---

## 3. 교육 목표

본 교과목의 교육 목표는 다음과 같다.

1. 생성형 AI를 활용하여 Python 코드를 생성할 수 있다.
2. AI가 생성한 코드를 이해하고 수정할 수 있다.
3. Google Colab에서 Python 코드를 실행하고 결과를 확인할 수 있다.
4. Python 기초 문법을 활용하여 간단한 문제를 해결할 수 있다.
5. AI와 협업하여 간단한 프로젝트를 완성할 수 있다.
6. AI 활용 과정에서 윤리적 사용과 자기 주도적 학습 태도를 기를 수 있다.

## 5. 수업 운영 방식

본 수업은 다음 흐름으로 운영한다.

```text
개념 소개
↓
AI Prompt 예시
↓
Google Colab 실습
↓
코드 실행 및 수정
↓
과제 수행
↓
프로젝트 적용
```

학생들은 매주 제공되는 교재와 실습 노트북을 읽고, Google Colab에서 코드를 실행한다.  
AI를 활용할 수 있으나, 사용한 Prompt와 수정 과정을 반드시 기록해야 한다.
