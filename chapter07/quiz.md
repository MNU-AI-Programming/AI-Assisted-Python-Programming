## 서술형 문제

### 14. 리스트가 필요한 이유를 설명하시오.

**예시 답안:**  
리스트는 여러 개의 값을 하나로 묶어 저장할 수 있기 때문에 필요하다.  
학생 이름이나 점수처럼 여러 데이터를 다룰 때 변수를 여러 개 만드는 것보다 리스트를 사용하는 것이 편리하다.  
반복문과 함께 사용하면 데이터를 하나씩 처리하기 쉽다.

---

### 15. CSV 파일을 저장한 뒤 다시 읽어 확인해야 하는 이유를 설명하시오.

**예시 답안:**  
파일 저장 코드가 오류 없이 실행되더라도 내용이 원하는 형태로 저장되었는지 확인해야 한다.  
특히 줄바꿈, 쉼표, 한글 인코딩 문제가 생길 수 있으므로 저장한 파일을 다시 읽어 확인하는 과정이 필요하다.
