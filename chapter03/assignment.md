# Chapter 3 과제
## AI와 함께 생활 계산기 만들기

---

## 과제 목표

이번 과제의 목표는 `input()`, 숫자 변환, 산술 연산자, f-string을 활용하여 생활 속 계산 프로그램을 만드는 것입니다.

---

## 과제 설명

ChatGPT를 활용하여 **생활 계산기 프로그램**을 작성하세요.

아래 주제 중 하나를 선택합니다.

1. BMI 계산기
2. 여행 경비 계산기
3. 카페 주문 금액 계산기
4. 용돈 계산기
5. 자유 주제 계산기

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. `input()` 함수를 2회 이상 사용할 것
3. `int()` 또는 `float()`을 사용하여 숫자로 변환할 것
4. 산술 연산자를 사용할 것
5. f-string을 사용하여 결과를 출력할 것
6. ChatGPT에 사용한 Prompt를 기록할 것
7. AI가 만든 코드를 자신의 주제에 맞게 수정할 것
8. 실행 결과를 함께 제출할 것

---

## 추천 Prompt 1: BMI 계산기

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Google Colab에서 실행할 수 있는 BMI 계산기 프로그램을 만들고 싶습니다.

키는 cm 단위로 입력받고, 몸무게는 kg 단위로 입력받아 BMI를 계산해 주세요.

input(), float(), f-string을 사용해 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

---

## 추천 Prompt 2: 여행 경비 계산기

```text
Python으로 여행 경비 계산기 프로그램을 만들어 주세요.

교통비, 숙박비, 식비, 기타 비용을 입력받고 총 경비를 계산하게 해 주세요.

input(), int() 또는 float(), f-string을 사용해 주세요.

프로그래밍 초보자도 이해할 수 있도록 주석을 달아 주세요.
```

---

## 추천 Prompt 3: 카페 주문 금액 계산기

```text
Python으로 카페 주문 금액 계산기를 만들고 싶습니다.

아메리카노 가격과 주문 수량을 입력받아 총 금액을 계산하게 해 주세요.

input(), int(), f-string을 사용해 주세요.

초보자가 이해할 수 있도록 코드 설명을 포함해 주세요.
```
