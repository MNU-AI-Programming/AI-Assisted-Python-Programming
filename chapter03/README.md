## 3.1 input() 함수

`input()` 함수는 사용자에게 값을 입력받을 때 사용합니다.

가장 기본적인 형태는 다음과 같습니다.

```python
name = input("이름을 입력하세요: ")
print(name)
```

실행하면 사용자가 값을 입력할 수 있습니다.

예를 들어 사용자가 `홍길동`이라고 입력하면 `name` 변수에 `"홍길동"`이 저장됩니다.

---

### 입력받은 값 출력하기

```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name, "님!")
```

실행 예시:

```text
이름을 입력하세요: 홍길동
안녕하세요, 홍길동 님!
```

## 3.2 input()의 결과는 문자열

중요한 점이 있습니다.

`input()`으로 입력받은 값은 기본적으로 **문자열(str)**입니다.

예를 들어 다음 코드를 실행해 봅시다.

```python
age = input("나이를 입력하세요: ")
print(type(age))
```

나이에 `20`을 입력해도 자료형은 문자열입니다.

```text
<class 'str'>
```

즉, 사용자가 숫자를 입력해도 Python은 처음에는 문자로 인식합니다.

---

### 왜 문제가 될까요?

다음 코드를 생각해 봅시다.

```python
age = input("나이를 입력하세요: ")
print(age + 1)
```

이 코드는 오류가 발생합니다.

이유는 `age`가 숫자가 아니라 문자열이기 때문입니다.

문자열과 숫자는 바로 더할 수 없습니다.

## 3.3 숫자로 변환하기

입력받은 값을 계산에 사용하려면 숫자로 변환해야 합니다.

정수로 변환할 때는 `int()`를 사용합니다.

```python
age = input("나이를 입력하세요: ")
age = int(age)

print(age + 1)
```

실수로 변환할 때는 `float()`을 사용합니다.

```python
height = input("키를 입력하세요: ")
height = float(height)

print(height)
```

---

### int()와 float() 비교

| 함수 | 의미 | 예시 |
|---|---|---|
| `int()` | 정수로 변환 | `20` |
| `float()` | 실수로 변환 | `175.5` |

---

## ⚠️ 자주 하는 실수 1: 숫자로 변환하지 않기

잘못된 코드:

```python
num = input("숫자를 입력하세요: ")
print(num + 10)
```

올바른 코드:

```python
num = input("숫자를 입력하세요: ")
num = int(num)
print(num + 10)
```

또는 한 줄로 작성할 수 있습니다.

```python
num = int(input("숫자를 입력하세요: "))
print(num + 10)
```

## 3.4 산술 연산자

Python에서는 다양한 계산을 할 수 있습니다.

| 연산자 | 의미 | 예시 | 결과 |
|---|---|---|---|
| `+` | 더하기 | `10 + 3` | `13` |
| `-` | 빼기 | `10 - 3` | `7` |
| `*` | 곱하기 | `10 * 3` | `30` |
| `/` | 나누기 | `10 / 3` | `3.333...` |
| `//` | 몫 | `10 // 3` | `3` |
| `%` | 나머지 | `10 % 3` | `1` |
| `**` | 거듭제곱 | `2 ** 3` | `8` |

---

### 계산 예제

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
```

## 3.5 f-string으로 계산 결과 출력하기

계산 결과를 보기 좋게 출력하려면 f-string을 사용하면 좋습니다.

```python
a = 10
b = 3

print(f"{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b}")
```

실행 결과:

```text
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.3333333333333335
```

---

### 소수점 자리 조절하기

계산 결과가 너무 길게 나올 때는 소수점 자리를 조절할 수 있습니다.

```python
result = 10 / 3
print(f"결과: {result:.2f}")
```

실행 결과:

```text
결과: 3.33
```

`:.2f`는 소수점 둘째 자리까지 출력하라는 뜻입니다.

## 3.6 BMI 계산기 만들기

BMI는 체질량지수입니다.

계산식은 다음과 같습니다.

```text
BMI = 몸무게(kg) / (키(m) * 키(m))
```

예를 들어 키가 1.75m이고 몸무게가 70kg이면 다음과 같이 계산합니다.

```python
height = 1.75
weight = 70

bmi = weight / (height * height)

print(bmi)
```

---

### 사용자 입력을 활용한 BMI 계산기

```python
height = float(input("키를 m 단위로 입력하세요: "))
weight = float(input("몸무게를 kg 단위로 입력하세요: "))

bmi = weight / (height * height)

print(f"당신의 BMI는 {bmi:.2f}입니다.")
```

---

## ⚠️ 자주 하는 실수 2: 키 단위 혼동

BMI 계산에서는 키를 **m 단위**로 사용합니다.

잘못된 예:

```text
키: 175
```

올바른 예:

```text
키: 1.75
```

만약 cm 단위로 입력받고 싶다면 m로 변환해야 합니다.

```python
height_cm = float(input("키를 cm 단위로 입력하세요: "))
height_m = height_cm / 100
```
