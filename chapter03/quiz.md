## 단답형 문제

### 11. 다음 코드의 실행 결과는?

```python

```

**정답:** 3

---

### 12. 다음 코드의 실행 결과는?

```python
print(10 % 3)
```

**정답:** 1

## 서술형 문제

### 14. `input()`으로 입력받은 값을 계산에 사용하기 전에 숫자로 변환해야 하는 이유를 설명하시오.

**예시 답안:**  
`input()`으로 입력받은 값은 기본적으로 문자열이기 때문이다.  
문자열은 숫자 계산에 바로 사용할 수 없으므로 `int()`나 `float()`을 사용하여 숫자로 변환해야 한다.

### 15. AI가 만든 계산 프로그램을 사용할 때 확인해야 할 점을 3가지 이상 쓰시오.

**예시 답안:**  
입력값을 숫자로 변환했는지 확인해야 한다.  
계산식이 맞는지 확인해야 한다.  
출력 결과가 실제 계산 결과와 일치하는지 확인해야 한다.  
사용자에게 입력 단위를 정확히 안내하는지도 확인해야 한다.
