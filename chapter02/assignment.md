# Chapter 2 과제
## AI와 함께 나의 프로필 카드 만들기

---

## 과제 목표

이번 과제의 목표는 Python의 `print()`, 변수, 자료형, f-string을 활용하여 자신의 정보를 출력하는 프로그램을 작성하는 것입니다.

---

## 과제 설명

ChatGPT를 활용하여 **나의 프로필 카드 프로그램**을 작성하세요.

프로그램은 다음 정보를 변수에 저장한 후 출력해야 합니다.

- 이름
- 학과 또는 전공
- 나이
- 관심 분야
- 좋아하는 것
- 이번 학기 목표

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. 변수 5개 이상 사용할 것
3. `print()` 함수 사용할 것
4. f-string을 2회 이상 사용할 것
5. `type()` 함수를 사용하여 변수 하나 이상의 자료형을 확인할 것
6. ChatGPT에 사용한 Prompt를 기록할 것
7. AI가 만든 코드를 자신의 정보에 맞게 수정할 것
8. 실행 결과를 함께 제출할 것

---

## 추천 Prompt

아래 Prompt를 ChatGPT에 입력해 봅니다.

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Python의 변수와 f-string을 사용하여 나의 프로필 카드 프로그램을 만들고 싶습니다.

이름, 학과, 나이, 관심 분야, 좋아하는 것, 이번 학기 목표를 변수에 저장하고 출력하는 예제 코드를 만들어 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```
