## 2.1 Python이란?

Python은 전 세계에서 널리 사용되는 프로그래밍 언어입니다.

Python은 문법이 비교적 쉽고 읽기 쉬워서 프로그래밍 입문자에게 많이 추천됩니다.

Python은 다음 분야에서 사용됩니다.

- 데이터 분석
- 인공지능
- 웹 개발
- 업무 자동화
- 교육
- 과학 계산
- 간단한 프로그램 제작

---

### Python이 쉬운 이유

Python은 사람의 언어와 비슷하게 읽을 수 있습니다.

예를 들어 화면에 문장을 출력하려면 다음과 같이 작성합니다.

```python
print("안녕하세요")
```

이 코드는 “안녕하세요”라는 문장을 화면에 출력하라는 뜻입니다.

## 2.3 print() 함수

`print()` 함수는 화면에 내용을 출력할 때 사용합니다.

가장 기본적인 형태는 다음과 같습니다.

```python
print("출력할 내용")
```

문장을 출력하려면 따옴표를 사용해야 합니다.

```python
print("안녕하세요")
print("Python을 배우고 있습니다")
print("AI와 함께 코딩합니다")
```

---

### 여러 줄 출력하기

`print()`를 여러 번 사용하면 여러 줄을 출력할 수 있습니다.

```python
print("이름: 홍길동")
print("학과: 교양학부")
print("관심 분야: 인공지능")
```

실행 결과:

```text
이름: 홍길동
학과: 교양학부
관심 분야: 인공지능
```

---

### 숫자 출력하기

숫자는 따옴표 없이 출력할 수 있습니다.

```python
print(10)
print(3.14)
```

하지만 숫자를 문장처럼 출력하고 싶다면 따옴표를 사용할 수도 있습니다.

```python
print("10")
print("3.14")
```

겉으로 보기에는 비슷하지만, Python 내부에서는 다르게 처리됩니다.

---

## ⚠️ 자주 하는 실수 1: 따옴표 빠뜨리기

```python

```

올바른 코드:

```python

```

문장을 출력할 때는 따옴표가 필요합니다.

## 2.4 변수

변수는 데이터를 저장하는 이름표입니다.

예를 들어 내 이름을 저장하고 싶다면 다음과 같이 작성합니다.

```python
name = "홍길동"
```

이 코드는 `"홍길동"`이라는 값을 `name`이라는 변수에 저장한다는 뜻입니다.

이후 `name`을 사용하면 저장된 값을 다시 사용할 수 있습니다.

```python
name = "홍길동"
print(name)
```

실행 결과:

```text
홍길동
```

---

### 변수는 왜 필요할까요?

변수를 사용하면 같은 값을 여러 번 쉽게 사용할 수 있습니다.

```python
name = "홍길동"

print(name)
print("안녕하세요,", name, "님")
print(name, "님, Python 수업을 시작합니다.")
```

만약 이름을 바꾸고 싶다면 변수에 저장된 값만 바꾸면 됩니다.

```python
name = "김민수"
```

---

### 변수 이름 규칙

변수 이름을 만들 때는 몇 가지 규칙이 있습니다.

| 규칙 | 예시 |
|---|---|
| 영어, 숫자, 밑줄 사용 가능 | `student_name` |
| 숫자로 시작할 수 없음 | `name1`은 가능, `1name`은 불가능 |
| 공백 사용 불가 | `student name`은 불가능 |
| 의미 있는 이름 사용 권장 | `age`, `score`, `name` |

## 2.5 자료형

자료형은 데이터의 종류를 의미합니다.

Python에서 자주 사용하는 기본 자료형은 다음과 같습니다.

| 자료형 | 의미 | 예시 |
|---|---|---|
| `int` | 정수 | `10`, `25`, `100` |
| `float` | 실수 | `3.14`, `175.5` |
| `str` | 문자열 | `"Hello"`, `"홍길동"` |
| `bool` | 참/거짓 | `True`, `False` |

---

### 숫자 자료형

정수는 소수점이 없는 숫자입니다.

```python
age = 20
print(age)
```

실수는 소수점이 있는 숫자입니다.

```python
height = 175.5
print(height)
```

---

### 문자열 자료형

문자열은 문자나 문장입니다.

문자열은 따옴표로 감싸야 합니다.

```python
name = "홍길동"
major = "컴퓨터학부"
print(name)
print(major)
```

---

### 불리언 자료형

불리언은 참 또는 거짓을 나타냅니다.

```python
is_student = True
print(is_student)
```

불리언은 나중에 조건문을 배울 때 중요하게 사용됩니다.

---

### type() 함수

`type()` 함수는 데이터의 자료형을 확인할 때 사용합니다.

```python
print(type(10))
print(type(3.14))
print(type("Hello"))
print(type(True))
```

실행 결과는 다음과 비슷합니다.

```text
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
```

## ⚠️ 자주 하는 실수 2: 숫자와 문자열 혼동

다음 두 값은 다릅니다.

```python
10
"10"
```

`10`은 숫자입니다.  
`"10"`은 문자열입니다.

숫자는 계산할 수 있지만, 문자열은 글자로 처리됩니다.

```python
print(10 + 5)
print("10" + "5")
```

실행 결과:

```text
15
105
```

## 2.6 문자열

문자열은 글자나 문장을 나타내는 자료형입니다.

Python에서는 문자열을 따옴표로 감쌉니다.

```python
message = "안녕하세요"
print(message)
```

---

### 문자열 연결하기

문자열은 `+`를 사용하여 연결할 수 있습니다.

```python
first = "AI"
second = "Coding"

print(first + second)
```

실행 결과:

```text
AICoding
```

띄어쓰기를 넣고 싶다면 공백 문자열을 추가합니다.

```python
print(first + " " + second)
```

실행 결과:

```text
AI Coding
```

---

### f-string 사용하기

문자열 안에 변수 값을 넣고 싶을 때는 f-string을 사용할 수 있습니다.

```python
name = "홍길동"
age = 20

print(f"안녕하세요. 저는 {name}입니다.")
print(f"제 나이는 {age}살입니다.")
```

실행 결과:

```text
안녕하세요. 저는 홍길동입니다.
제 나이는 20살입니다.
```

f-string은 앞으로 자주 사용하게 됩니다.

## 2.7 AI가 만든 코드 수정하기

AI에게 다음과 같은 프로그램을 요청했다고 합시다.

```text
이름과 나이를 변수에 저장하고 자기소개를 출력하는 Python 코드를 만들어 주세요.
```

AI는 다음과 비슷한 코드를 만들 수 있습니다.

```python
name = "홍길동"
age = 20

print(f"안녕하세요. 저는 {name}입니다.")
print(f"저는 {age}살입니다.")
```

이 코드를 그대로 사용하는 것이 아니라, 자신의 정보에 맞게 수정해야 합니다.

예:

```python
name = "김하늘"
age = 19

print(f"안녕하세요. 저는 {name}입니다.")
print(f"저는 {age}살입니다.")
```
