# AI활용코딩연습(ZBE135) 15주 직접 녹화 가이드

## 고정 원칙

- 한 주에 영상 파일 1개
- 모든 주차는 독립형으로 구성하고 오프닝 직후 학습목표로 시작
- 영상 길이 25분 이상, 권장 26~28분
- GitHub가 원본 교재이며 영상은 GitHub 자료를 설명
- 슬라이드 15장과 Colab Level 1~4를 같은 순서로 진행
- AI 답변보다 실행 증거·테스트·수정 이유를 강조

## 권장 녹화 환경

- 해상도: 1920×1080, 30fps
- 마이크: 48kHz, 피크 -12~-6dB
- 브라우저 확대: 110~125%
- Colab 코드 글자: 17px 이상
- 알림·메일·메신저 끄기
- 개인정보·API 키·Drive 목록 가리기
