## 5.1 반복문이란?

반복문은 같은 작업을 여러 번 실행할 때 사용하는 문법입니다.

예를 들어 다음 작업을 생각해 봅시다.

```text
안녕하세요를 5번 출력하기
1부터 10까지 출력하기
구구단 2단 출력하기
사용자가 정답을 맞힐 때까지 계속 입력받기
```

이런 작업은 반복문을 사용하면 훨씬 간단하게 작성할 수 있습니다.

---

### 반복문이 없는 코드

```python
print("안녕하세요")
print("안녕하세요")
print("안녕하세요")
print("안녕하세요")
print("안녕하세요")
```

### 반복문을 사용한 코드

```python
for i in range(5):
    print("안녕하세요")
```

반복문을 사용하면 같은 코드를 여러 번 쓰지 않아도 됩니다.

---

## 5.2 for 반복문

`for` 반복문은 정해진 횟수만큼 반복할 때 많이 사용합니다.

기본 구조는 다음과 같습니다.

```python
for 변수 in 반복범위:
    반복할 코드
```

예를 들어 5번 반복하려면 다음과 같이 작성합니다.

```python
for i in range(5):
    print("안녕하세요")
```

---

### range() 함수

`range()`는 숫자의 범위를 만들어 주는 함수입니다.

```python
range(5)
```

이는 0부터 4까지의 숫자를 의미합니다.

즉, 총 5번 반복됩니다.

```python
for i in range(5):
    print(i)
```

실행 결과:

```text
0
1
2
3
4
```

---

### 1부터 5까지 출력하기

```python
for i in range(1, 6):
    print(i)
```

실행 결과:

```text
1
2
3
4
5
```

`range(1, 6)`은 1부터 5까지를 의미합니다.  
끝 숫자인 6은 포함되지 않습니다.

---

## ⚠️ 자주 하는 실수 1: range의 끝 숫자 착각

```python
for i in range(1, 5):
    print(i)
```

실행 결과:

```text
1
2
3
4
```

`range(1, 5)`는 1부터 5까지가 아니라 1부터 4까지입니다.

## 5.3 for 반복문으로 합계 구하기

반복문은 숫자의 합계를 구할 때도 사용할 수 있습니다.

예를 들어 1부터 10까지의 합계를 구해 봅시다.

```python
total = 0

for i in range(1, 11):
    total = total + i

print(total)
```

실행 결과:

```text
55
```

---

### 코드 이해하기

```python
total = 0
```

합계를 저장할 변수를 0으로 시작합니다.

```python
for i in range(1, 11):
```

1부터 10까지 반복합니다.

```python
total = total + i
```

기존 합계에 현재 숫자 `i`를 더합니다.

---

### 더 짧은 표현

```python
total = total + i
```

는 다음과 같이 쓸 수도 있습니다.

```python
total += i
```

두 코드는 같은 의미입니다.

## 5.4 구구단 만들기

반복문을 배우면 구구단을 쉽게 만들 수 있습니다.

2단을 출력해 봅시다.

```python
dan = 2

for i in range(1, 10):
    print(f"{dan} x {i} = {dan * i}")
```

실행 결과:

```text
2 x 1 = 2
2 x 2 = 4
...
2 x 9 = 18
```

---

### 사용자에게 단 입력받기

```python
dan = int(input("출력할 단을 입력하세요: "))

for i in range(1, 10):
    print(f"{dan} x {i} = {dan * i}")
```

사용자가 원하는 단을 입력하면 해당 단이 출력됩니다.

## 5.5 while 반복문

`while` 반복문은 조건이 참인 동안 계속 반복합니다.

기본 구조는 다음과 같습니다.

```python
while 조건:
    반복할 코드
```

예를 들어 1부터 5까지 출력하려면 다음과 같이 작성할 수 있습니다.

```python
count = 1

while count <= 5:
    print(count)
    count = count + 1
```

---

### while 문에서 중요한 점

`while` 문은 조건이 계속 참이면 끝나지 않습니다.

따라서 반복이 끝날 수 있도록 조건을 바꾸는 코드가 필요합니다.

```python
count = count + 1
```

이 코드가 없으면 `count`가 계속 1이므로 무한 반복될 수 있습니다.

---

## ⚠️ 자주 하는 실수 2: 무한 반복

잘못된 코드:

```python
count = 1

while count <= 5:
    print(count)
```

이 코드는 `count`가 증가하지 않기 때문에 계속 반복됩니다.

올바른 코드:

```python
count = 1

while count <= 5:
    print(count)
    count = count + 1
```

---

## 5.6 반복문과 조건문 함께 사용하기

반복문 안에서 조건문을 사용할 수 있습니다.

예를 들어 1부터 10까지 숫자 중 짝수만 출력해 봅시다.

```python
for i in range(1, 11):
    if i % 2 == 0:
        print(i)
```

`i % 2 == 0`은 `i`를 2로 나누었을 때 나머지가 0인지 확인하는 조건입니다.

즉, 짝수인지 확인합니다.

---

### 홀수 출력하기

```python
for i in range(1, 11):
    if i % 2 == 1:
        print(i)
```

## 5.7 숫자 맞추기 게임 만들기

이번에는 `while` 반복문을 사용하여 숫자 맞추기 게임을 만들어 봅시다.

```python
answer = 7
guess = 0

while guess != answer:
    guess = int(input("숫자를 맞혀 보세요: "))

    if guess == answer:
        print("정답입니다!")
    else:
        print("틀렸습니다. 다시 시도해 보세요.")
```

이 프로그램은 사용자가 정답을 맞힐 때까지 계속 입력을 받습니다.

---

### 조금 더 친절한 숫자 맞추기 게임

```python
answer = 7
guess = 0

while guess != answer:
    guess = int(input("1부터 10 사이의 숫자를 입력하세요: "))

    if guess == answer:
        print("정답입니다!")
    elif guess < answer:
        print("더 큰 숫자입니다.")
    else:
        print("더 작은 숫자입니다.")
```

## 5.8 random 모듈 사용하기

매번 정답이 7이면 게임이 재미없습니다.

Python의 `random` 모듈을 사용하면 임의의 숫자를 만들 수 있습니다.

```python
import random

answer = random.randint(1, 10)
```

`random.randint(1, 10)`은 1부터 10 사이의 정수 중 하나를 무작위로 선택합니다.

---

### random을 사용한 숫자 맞추기 게임

```python
import random

answer = random.randint(1, 10)
guess = 0

while guess != answer:
    guess = int(input("1부터 10 사이의 숫자를 입력하세요: "))

    if guess == answer:
        print("정답입니다!")
    elif guess < answer:
        print("더 큰 숫자입니다.")
    else:
        print("더 작은 숫자입니다.")
```

## 5.9 break 사용하기

`break`는 반복문을 중간에 종료할 때 사용합니다.

예를 들어 사용자가 `0`을 입력하면 종료하는 프로그램을 만들어 봅시다.

```python
while True:
    number = int(input("숫자를 입력하세요. 종료하려면 0 입력: "))

    if number == 0:
        print("프로그램을 종료합니다.")
        break

    print(f"입력한 숫자: {number}")
```

`while True`는 계속 반복하라는 뜻입니다.  
하지만 `break`를 만나면 반복문이 종료됩니다.

---

## ⚠️ 자주 하는 실수 3: break 없이 while True 사용

`while True`를 사용할 때는 반복문을 멈출 수 있는 조건이 있어야 합니다.

```python
if number == 0:
    break
```

이런 코드가 없다면 프로그램이 계속 반복될 수 있습니다.
