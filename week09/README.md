수치 데이터가 많아지면 NumPy 배열의 벡터 연산이 반복문보다 간결하고 일관됩니다.

**문제 상황:** 점수 배열의 평균·표준편차·총점을 계산하고 80점 이상 값을 선택합니다.

## 6. 주요 코드 예제

```python
import numpy as np
scores=np.array([85,92,76,88,95])
print(scores.shape,scores.dtype,scores.mean(),scores.std())
print(scores[scores>=80])
```
