# CONTRIBUTING

이 저장소는 **AI 활용 코딩** 수업을 위한 공개 교재 저장소입니다.

교재 수정, 오타 제보, 실습 코드 개선, 과제 보완 제안을 환영합니다.

---

## 기여할 수 있는 내용

다음과 같은 기여가 가능합니다.

- 오타 수정
- 설명 보완
- 더 쉬운 예제 제안
- Google Colab 실습 오류 수정
- 과제 문항 개선
- 퀴즈 추가
- Prompt 예시 추가
- 슬라이드 개선
- 학생 FAQ 추가

---

## 문서 작성 원칙

문서는 다음 원칙을 따릅니다.

1. 비전공자도 이해할 수 있도록 쉽게 작성합니다.
2. 어려운 용어는 처음 등장할 때 설명합니다.
3. Python 코드는 Google Colab에서 실행 가능해야 합니다.
4. AI Prompt는 학생이 그대로 복사해 사용할 수 있도록 작성합니다.
5. AI가 만든 코드는 반드시 검토하고 수정합니다.
6. 너무 복잡한 문법은 피합니다.

---

## 파일 구조

```text
chapterXX/
├── README.md       # 교재 본문
├── practice.ipynb  # Colab 실습
├── slides.md       # Marp 슬라이드
├── assignment.md   # 과제
└── quiz.md         # 퀴즈
```

## Pull Request 작성 시 확인 사항

Pull Request를 보내기 전에 다음을 확인해 주세요.

- Markdown 문법이 깨지지 않는가?
- 코드 블록 언어가 명시되어 있는가?
- Python 코드는 Colab에서 실행되는가?
- 학생 수준에 맞는 설명인가?
- 불필요하게 어려운 용어가 없는가?
- AI 사용 관련 윤리 원칙에 어긋나지 않는가?

---

## Issue 작성 예시

```markdown
## 위치
chapter03/README.md

## 문제
BMI 계산기 설명에서 cm와 m 단위 설명이 부족합니다.

## 제안
cm 단위로 입력받을 때는 100으로 나누어 m 단위로 변환한다는 설명을 추가하면 좋겠습니다.
```

## 학생 기여

학생도 다음 방식으로 기여할 수 있습니다.

- 실습 오류 제보
- 이해하기 어려운 문장 표시
- 새로운 예제 제안
- 자주 묻는 질문 추가
- 프로젝트 예시 공유

단, 과제 정답 전체를 공개적으로 올리는 것은 피합니다.
