## 교과목 소개

**AI활용코딩연습(ZBE135)** 은 생성형 AI를 단순한 정답 생성기가 아니라 코드를 함께 설계하고 검증하는 **학습 파트너**로 활용하는 1학점 집중수업입니다.

학생은 Python 핵심 문법을 복습한 뒤 자료구조, 함수, 파일 처리, 객체지향 프로그래밍, NumPy, pandas, 데이터 시각화를 실습합니다. 모든 과정에서 AI가 제안한 코드를 직접 읽고 실행하며, 오류와 한계를 확인하고 수정 근거를 기록합니다. 마지막에는 실행 가능한 데이터 분석 미니 프로젝트를 완성합니다.

## 학습 목표

수강생은 수업을 마친 뒤 다음을 수행할 수 있습니다.

1. Python의 조건문, 반복문, 자료구조와 함수를 조합해 문제를 해결합니다.
2. 파일·CSV·예외 처리와 클래스를 활용해 프로그램을 구조화합니다.
3. NumPy와 pandas로 데이터를 불러오고 정제·변환·집계합니다.
4. matplotlib으로 분석 결과를 시각화하고 의미를 설명합니다.
5. 생성형 AI가 제안한 코드의 정확성, 안전성, 가독성을 테스트하고 개선합니다.
6. 데이터, 분석 과정, 결론과 한계가 포함된 재현 가능한 미니 프로젝트를 완성합니다.

## 실습 진행 방법

모든 실습은 다음 네 단계로 진행합니다.

1. **Level 1 따라하기:** 코드를 실행하기 전에 결과를 예상합니다.
2. **Level 2 바꾸기:** 값이나 조건을 두 곳 이상 변경하고 결과 차이를 설명합니다.
3. **Level 3 적용하기:** 여러 개념을 조합하고 정상·경계·오류 사례로 검증합니다.
4. **Level 4 확장하기:** 필요한 기능을 직접 설계하고 AI 제안, 채택·거절 이유, 수정 내용과 테스트 결과를 기록합니다.

## 생성형 AI 활용 원칙

- AI가 만든 코드는 그대로 제출하지 않고 반드시 직접 읽고 실행합니다.
- 사용한 프롬프트, 채택한 제안, 거절한 제안, 직접 수정한 부분과 이유를 기록합니다.
- 정상 사례뿐 아니라 경계값과 오류 사례도 함께 테스트합니다.
- AI의 설명과 코드가 과제 요구사항 및 실제 실행 결과와 일치하는지 확인합니다.
- 개인정보, 비밀번호, API 키와 타인의 결과물을 AI 또는 GitHub에 입력하지 않습니다.
- 최종 결과에 대한 책임은 AI가 아니라 제출자에게 있습니다.
