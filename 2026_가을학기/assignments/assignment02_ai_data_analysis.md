## 2. 과제 목적

이 과제는 pandas와 matplotlib을 이용하여 데이터를 정제·탐색·시각화하고, 분석 질문에 근거하여 결론을 설명하는 미니 프로젝트입니다. 생성형 AI는 코드 작성과 오류 해결에 활용할 수 있지만, 학생은 AI가 만든 코드의 정확성을 직접 검증하고 수정 근거를 제시해야 합니다.

## 3. 프로젝트 주제

다음 중 하나를 선택합니다.

### 선택 A. 지출 데이터 분석

`expenses.csv`를 이용하여 개인 또는 집단의 지출 패턴을 분석합니다.

분석 질문 예시:

- 지출이 가장 많은 항목은 무엇인가?
- 날짜 또는 요일에 따라 지출 패턴이 달라지는가?
- 평균보다 매우 큰 지출은 어떤 항목에서 나타나는가?
- 예산을 줄이려면 어떤 항목을 우선 확인해야 하는가?

### 선택 B. 설문 데이터 분석

`survey.csv`를 이용하여 응답자의 특성과 응답 경향을 분석합니다.

분석 질문 예시:

- 결측치와 중복 응답은 얼마나 존재하는가?
- 집단별 응답 평균이나 분포는 어떻게 다른가?
- 가장 긍정적 또는 부정적인 응답 항목은 무엇인가?
- 데이터만으로 결론 내리기 어려운 부분은 무엇인가?

## 4. 필수 수행 단계

### 단계 1. 분석 질문 설정

- 데이터로 확인할 수 있는 구체적인 질문을 **2개 이상** 작성합니다.
- 질문을 선택한 이유와 예상 결과를 설명합니다.
- 단순히 “데이터를 분석한다”가 아니라 비교 대상과 측정값이 드러나야 합니다.

### 단계 2. 데이터 불러오기와 구조 확인

다음 항목을 코드와 설명으로 제시합니다.

- 행과 열의 수
- 열 이름과 자료형
- 기초 통계량
- 결측치 수
- 중복 행 수
- 이상하거나 분석에 사용할 수 없는 값

### 단계 3. 데이터 정제

다음 중 데이터에 필요한 작업을 수행하고 이유를 설명합니다.

- 결측치 제거 또는 대체
- 중복 행 제거
- 문자열·날짜·숫자 자료형 변환
- 범주 이름 통일
- 이상값 확인 및 처리

정제 전후의 행 수, 결측치 또는 주요 통계가 어떻게 변했는지 비교표를 만듭니다.

### 단계 4. 데이터 분석

- pandas의 필터링, 정렬, 그룹화와 집계 기능을 사용합니다.
- 분석 질문마다 계산 결과를 표로 제시합니다.
- 코드 실행 결과가 질문에 답하는 이유를 Markdown으로 설명합니다.
- 최소 한 번 이상 직접 작성한 함수를 분석 과정에 사용합니다.

### 단계 5. 데이터 시각화

서로 다른 목적의 차트를 **3개 이상** 작성합니다.

- 범주 비교 차트 1개 이상
- 분포 또는 시간 변화 차트 1개 이상
- 학생이 선택한 추가 차트 1개 이상

모든 차트에는 다음 요소가 있어야 합니다.

- 의미가 드러나는 제목
- 축 이름과 단위
- 필요한 경우 범례
- 차트 아래 2문장 이상의 해석

### 단계 6. 결론과 한계

- 분석 질문별 결론을 데이터 근거와 함께 작성합니다.
- 실생활에서 적용할 수 있는 제안 한 가지를 작성합니다.
- 데이터의 크기, 수집 방법, 결측치 또는 변수 부족 등 분석의 한계를 두 가지 이상 설명합니다.

### 단계 7. 재현성 확인

- 런타임을 다시 시작하고 모든 셀을 순서대로 실행합니다.
- 파일 경로를 개인 컴퓨터의 절대 경로로 작성하지 않습니다.
- 데이터 불러오기부터 결론까지 한 번에 재실행되는지 확인합니다.

## 5. AI 코드 검증 활동

AI가 작성하거나 수정한 코드 중 하나를 선택하여 다음 표를 작성합니다.

| 항목 | 기록 내용 |
|---|---|
| 사용한 프롬프트 | AI에 실제 입력한 내용 |
| AI의 제안 | 제안한 코드 또는 분석 방법 요약 |
| 위험 또는 의심 지점 | 잘못된 열 이름, 자료형, 계산, 그래프 해석 가능성 등 |
| 검증 방법 | 직접 계산, 작은 테스트 데이터, pandas 결과 비교 등 |
| 직접 수정 | 변경 전·후 코드와 수정 이유 |
| 최종 판단 | 채택, 일부 채택 또는 거절 |

다음 중 하나 이상의 검증을 반드시 수행합니다.

- 5행 이하의 작은 테스트 데이터로 결과를 손으로 계산하여 비교
- 같은 값을 서로 다른 pandas 명령으로 계산하여 비교
- 정제 전후 행 수와 결측치 수가 논리적으로 일치하는지 확인
- 그래프에 사용한 집계값과 표의 집계값이 같은지 확인

권장 프롬프트 예시:

```text
다음 pandas 코드가 분석 질문에 올바르게 답하는지 검토해 줘.
완성 코드를 다시 작성하기 전에 자료형, 결측치, 그룹화 기준, 집계 방식에서 발생할 수 있는 오류를 먼저 목록으로 제시해 줘.
그리고 각 오류를 확인할 수 있는 작은 테스트 방법을 설명해 줘.
```
