## 2. 과제 목적

이 과제는 Python의 자료구조, 조건문, 반복문, 함수, 파일·CSV 처리와 예외 처리를 하나의 프로그램에 통합하는 연습입니다. 단순히 실행되는 코드를 만드는 데서 끝나지 않고, 정상 입력과 경계값을 테스트하여 결과가 정확한지 설명해야 합니다.

## 3. 문제 상황

교수자는 학생 성적이 저장된 CSV 파일을 입력받아 다음 정보를 자동으로 계산하는 프로그램이 필요합니다.

- 전체 학생 수
- 과목별 또는 총점의 평균, 최댓값, 최솟값
- 학생별 총점과 평균
- 평균을 기준으로 한 등급
- 우수 학생과 재학습이 필요한 학생 목록
- 분석 결과를 저장한 새로운 CSV 파일

## 4. 필수 구현 요구사항

### 기능 1. CSV 데이터 불러오기

- `scores.csv`를 읽어 학생 데이터를 저장합니다.
- 숫자 열은 계산 가능한 자료형으로 변환합니다.
- 파일이 없거나 숫자로 변환할 수 없는 값이 있을 때 이해하기 쉬운 오류 메시지를 출력합니다.

### 기능 2. 학생별 점수 계산

- 각 학생의 총점과 평균을 계산합니다.
- 다음 기준으로 등급을 부여합니다.

| 평균 | 등급 |
|---:|:---:|
| 90점 이상 | A |
| 80점 이상 90점 미만 | B |
| 70점 이상 80점 미만 | C |
| 60점 이상 70점 미만 | D |
| 60점 미만 | F |

### 기능 3. 전체 통계 계산

- 전체 평균, 최댓값, 최솟값을 계산합니다.
- 평균이 가장 높은 학생을 출력합니다.
- 평균 90점 이상인 우수 학생과 60점 미만인 재학습 대상 학생을 각각 출력합니다.

## 5. 필수 테스트

다음 사례를 모두 테스트하고 예상 결과와 실제 결과를 표로 기록합니다.

| 테스트 | 확인할 내용 |
|---|---|
| 정상 사례 | 여러 학생의 총점·평균·등급이 정확한가? |
| 경계값 1 | 평균이 정확히 90점일 때 A인가? |
