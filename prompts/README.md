# AI Prompt 모음

> 이 폴더는 AI 활용 코딩 수업에서 사용할 수 있는 Prompt 예시를 제공합니다.  
> 학생들은 Prompt를 그대로 복사해 사용할 수 있지만, 자신의 상황에 맞게 수정하는 것이 중요합니다.

---

## Prompt 사용 원칙

좋은 Prompt에는 다음 요소가 포함되면 좋습니다.

1. 역할 부여
2. 학습자 수준
3. 원하는 작업
4. 출력 형식
5. 조건
6. 예제 요청

---

## 기본 구조

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 비전공 대학생입니다.

[배우고 싶은 내용]을 쉽게 설명해 주세요.

Google Colab에서 실행할 수 있는 예제를 만들어 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

## 주의사항

AI의 답변이 항상 정답은 아닙니다.

다음을 반드시 확인하세요.

- 코드가 실행되는가?
- 오류가 없는가?
- 내가 이해할 수 있는가?
- 과제 조건을 만족하는가?
- 내 상황에 맞게 수정했는가?
