# 보고서 작성 Prompt

---

## 1. 과제 보고서 정리

```text
아래 내용을 바탕으로 Python 실습 과제 보고서를 정리해 주세요.

보고서 형식:
1. 과제 제목
2. 사용한 AI Prompt
3. 최종 코드 설명
4. 실행 결과 설명
5. 직접 수정한 부분
6. 느낀 점

내용:
[여기에 내 기록 붙여 넣기]

문체는 대학생 과제 제출용으로 자연스럽게 작성해 주세요.
```

---

## 2. 프로젝트 결과 보고서 작성

```text
아래 내용을 바탕으로 최종 프로젝트 결과 보고서를 작성해 주세요.

보고서 형식:
1. 프로젝트 제목
2. 프로젝트 목적
3. 주요 기능
4. 사용한 Python 문법
5. 사용한 AI Prompt
6. 최종 코드 설명
7. 실행 결과
8. 오류와 해결 과정
9. 직접 수정한 부분
10. 느낀 점

내용:
[여기에 프로젝트 기록 붙여 넣기]

비전공 대학생 수준의 자연스러운 문체로 작성해 주세요.
```

---

## 3. 발표 대본 작성

```text
아래 프로젝트 내용을 바탕으로 3분 발표 대본을 작성해 주세요.

발표 내용:
1. 프로젝트 제목
2. 만든 이유
3. 주요 기능
4. AI 활용 방법
5. 오류와 해결 과정
6. 직접 수정한 부분
7. 배운 점

내용:
[여기에 프로젝트 기록 붙여 넣기]

말로 발표하기 쉽게 자연스럽게 작성해 주세요.
```

실습 내용:
[여기에 실습 내용 붙여 넣기]
