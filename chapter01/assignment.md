# Chapter 1 과제
## AI와 함께 자기소개 프로그램 만들기

---

## 과제 목표

이번 과제의 목표는 생성형 AI를 활용하여 간단한 Python 프로그램을 만들고, Google Colab에서 실행해 보는 것입니다.

---

## 과제 설명

ChatGPT를 활용하여 **자기소개 프로그램**을 작성하세요.

프로그램은 다음 정보를 출력해야 합니다.

- 이름
- 학과 또는 전공
- 관심 분야
- 이 수업에서 배우고 싶은 것
- AI를 활용해 보고 싶은 일

---

## 필수 조건

다음 조건을 모두 만족해야 합니다.

1. Google Colab에서 작성하고 실행할 것
2. `print()` 함수를 사용할 것
3. 최소 5줄 이상 출력할 것
4. AI에게 사용한 Prompt를 함께 기록할 것
5. AI가 만든 코드를 그대로 제출하지 말고, 최소 1곳 이상 직접 수정할 것
6. 실행 결과 화면을 제출할 것

---

## 추천 Prompt

아래 Prompt를 ChatGPT에 입력해 봅니다.

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Python의 print() 함수를 사용하여 자기소개 프로그램을 만들고 싶습니다.

이름, 학과, 관심 분야, 이 수업에서 배우고 싶은 것, AI를 활용해 보고 싶은 일을 출력하는 예제 코드를 만들어 주세요.

각 줄에 주석을 달아 주세요.
```
