# 학습 목표

- AI가 무엇인지 설명할 수 있다.
- 생성형 AI가 무엇인지 설명할 수 있다.

- ChatGPT는 사람처럼 생각할까요?

- AI가 코드를 만들어 준다면 우리는 왜 코딩을 배워야 할까요?

# AI란 무엇인가?

AI는 Artificial Intelligence의 약자입니다.

우리말로는 인공지능입니다.

사람이 하는 지적인 활동을 컴퓨터가 수행하도록 만든 기술입니다.

---

# 우리 주변의 AI

|---|---|
| YouTube | 영상 추천 |
| Netflix | 영화 추천 |

---

# 생성형 AI란?

생성형 AI는 새로운 결과물을 만들어 내는 AI입니다.

- 글쓰기
- 번역
- 이미지 생성
- 음악 생성
- Python 코드 작성

---

# AI와 함께 코딩하는 시대

기존 방식

문제 이해 → 코드 작성 → 실행 → 오류 수정

AI 활용 방식

문제 이해 → AI에게 질문 → 코드 생성 → 이해 → 실행 → 수정

---

# ChatGPT는 무엇을 도와줄 수 있나?

- 개념 설명
- 코드 작성
- 코드 설명
- 오류 수정
- 프로그램 개선
- 아이디어 제안

---

# AI는 완벽하지 않다

AI는 다음과 같은 실수를 할 수 있습니다.

- 틀린 코드를 작성할 수 있음
- 오래된 정보를 말할 수 있음
- 실행되지 않는 코드를 만들 수 있음
- 질문을 잘못 이해할 수 있음

---

# Prompt란?

AI에게 전달하는 질문이나 명령입니다.

좋은 Prompt는 좋은 결과를 만듭니다.

---

# 나쁜 Prompt

```text
파이썬 알려줘
```

너무 넓고 구체적이지 않습니다.

---

# 좋은 Prompt

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

print() 함수를 쉬운 예제와 함께 설명해 주세요.
예제는 3개 만들어 주세요.
```

---

# 좋은 Prompt의 요소

1. 역할
2. 대상
3. 작업
4. 형식
5. 조건

---

# Google Colab

웹 브라우저에서 Python 코드를 실행할 수 있는 온라인 실습 환경입니다.

- 설치 불필요
- Python 설치 불필요
- Google Drive 저장 가능
- 공유 가능

---

# 첫 번째 Python 코드

```python
print("Hello, World!")
```

실행 결과

```text
Hello, World!
```

---

# 코드 이해하기

```python
print("Hello, World!")
```

- `print()`는 출력 함수
- `"Hello, World!"`는 출력할 문장
- 따옴표 안의 문장은 문자열

---

# AI에게 코드 설명 요청하기

```text
아래 Python 코드를 한 줄씩 설명해 주세요.

print("Hello, World!")
```

---

# 실습

1. Hello, World 출력하기
2. 이름 입력받기
3. 자기소개 출력하기
4. AI가 만든 코드 실행하기
5. 오류 수정하기

---

# 자주 하는 실수

```python
print(Hello)
```

문자열에는 따옴표가 필요합니다.

```python
print("Hello")
```

---

# 핵심 정리

- AI는 학습 파트너입니다.
- 생성형 AI는 글과 코드를 만들 수 있습니다.
- 좋은 Prompt가 좋은 답변을 만듭니다.
- Colab을 사용하면 설치 없이 Python을 실행할 수 있습니다.
- AI가 만든 코드는 반드시 이해하고 실행해야 합니다.
