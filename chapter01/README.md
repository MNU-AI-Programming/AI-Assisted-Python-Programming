## 1.1 AI란 무엇인가?

AI는 **Artificial Intelligence**의 약자로, 우리말로는 **인공지능**이라고 합니다.

인공지능은 사람이 하는 지적인 활동을 컴퓨터가 할 수 있도록 만든 기술입니다.

예를 들어 사람은 다음과 같은 일을 할 수 있습니다.

- 사진을 보고 고양이와 강아지를 구분한다.
- 길을 찾는다.
- 질문에 답한다.
- 글을 쓴다.
- 말을 이해한다.
- 문제를 해결한다.

AI는 이러한 일을 컴퓨터가 수행할 수 있도록 만든 기술입니다.

---

### 우리 주변의 AI

AI는 이미 우리 주변에서 많이 사용되고 있습니다.

| 서비스 | AI가 하는 일 |
|---|---|
| YouTube | 관심 있을 만한 영상을 추천 |
| Netflix | 좋아할 만한 영화 추천 |
| 네이버 검색 | 관련성이 높은 검색 결과 제공 |
| Papago | 문장 번역 |
| 스마트폰 카메라 | 얼굴 인식 |
| ChatGPT | 질문에 대한 답변 생성 |

우리는 이미 매일 AI를 사용하고 있습니다.  
다만 그것을 AI라고 의식하지 못했을 뿐입니다.

## 1.2 생성형 AI란?

생성형 AI는 새로운 결과물을 만들어 내는 AI입니다.

기존 AI가 주로 분류하거나 예측하는 일을 했다면, 생성형 AI는 새로운 내용을 만들어 냅니다.

생성형 AI가 만들 수 있는 것은 다양합니다.

- 글
- 그림
- 음악
- 발표 자료
- 요약문
- 번역문
- Python 코드
- 보고서 초안

ChatGPT는 대표적인 생성형 AI입니다.

---

### 생성형 AI와 기존 AI의 차이

| 구분 | 기존 AI | 생성형 AI |
|---|---|---|
| 주요 기능 | 분류, 예측, 인식 | 생성, 작성, 변환 |
| 예시 | 스팸 메일 분류 | 이메일 초안 작성 |
| 결과 | 정답 또는 분류값 | 새로운 문장, 코드, 이미지 |
| 활용 | 추천, 인식, 예측 | 글쓰기, 코딩, 디자인 |

## 1.3 AI와 함께 코딩하는 시대

과거에는 프로그램을 만들기 위해 사람이 처음부터 끝까지 코드를 작성해야 했습니다.

하지만 지금은 AI에게 만들고 싶은 프로그램을 설명하면 AI가 코드의 초안을 만들어 줍니다.

이때 중요한 것은 AI가 만든 코드를 그대로 믿는 것이 아닙니다.

사람은 다음과 같은 역할을 해야 합니다.

- 문제를 정확히 설명한다.
- AI가 만든 코드를 이해한다.
- 코드를 실행해 본다.
- 오류를 수정한다.
- 더 좋은 코드로 개선한다.
- 자신의 목적에 맞게 바꾼다.

---

### 기존 코딩 방식

```text
문제 이해
↓
문법 기억
↓
코드 작성
↓
실행
↓
오류 수정
```

### AI 활용 코딩 방식

```text
문제 이해
↓
AI에게 질문
↓
AI가 코드 생성
↓
코드 이해
↓
Google Colab에서 실행
↓
수정 및 개선
```

---

## 💡 AI Tip

AI에게 질문할 때는 "무엇을 만들고 싶은지"를 구체적으로 설명해야 합니다.

나쁜 예:

```text
코드 만들어줘
```

좋은 예:

```text
Python으로 이름을 입력하면 "안녕하세요, 홍길동님!"처럼 출력하는 프로그램을 만들어 주세요.
프로그래밍을 처음 배우는 대학생도 이해할 수 있도록 코드에 주석을 달아 주세요.
```

## 1.4 ChatGPT 시작하기

ChatGPT는 사용자의 질문을 이해하고 자연스러운 답변을 만들어 주는 생성형 AI입니다.

이 교재에서는 ChatGPT를 다음과 같이 활용합니다.

- 개념 설명 요청
- 코드 작성 요청
- 코드 설명 요청
- 오류 해결 요청
- 프로그램 개선 요청
- 과제 아이디어 도출

---

### ChatGPT는 완벽하지 않습니다

ChatGPT가 항상 맞는 답을 주는 것은 아닙니다.

다음과 같은 문제가 발생할 수 있습니다.

- 틀린 코드를 작성할 수 있습니다.
- 오래된 정보를 말할 수 있습니다.
- 실행되지 않는 코드를 만들 수 있습니다.
- 질문을 잘못 이해할 수 있습니다.
- 그럴듯하지만 틀린 설명을 할 수 있습니다.

따라서 AI가 만든 코드는 반드시 직접 실행하고 확인해야 합니다.

## 1.5 좋은 Prompt 작성법

AI에게 전달하는 질문이나 명령을 **Prompt**라고 합니다.

좋은 Prompt는 좋은 결과를 만듭니다.

---

### 나쁜 Prompt

```text
파이썬 알려줘
```

이 질문은 너무 넓습니다. AI는 무엇을 얼마나 설명해야 할지 알기 어렵습니다.

---

### 보통 Prompt

```text
Python의 print() 함수를 설명해 주세요.
```

이 질문은 앞의 질문보다 좋지만, 학습자의 수준이나 원하는 형식이 부족합니다.

---

### 좋은 Prompt

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

print() 함수를 쉬운 예제와 함께 설명해 주세요.

예제는 3개 만들어 주세요.

각 예제에는 코드 설명을 한 줄씩 붙여 주세요.
```

좋은 Prompt에는 다음이 포함됩니다.

1. 역할: AI가 어떤 입장에서 답해야 하는가?
2. 대상: 누가 이해해야 하는가?
3. 작업: 무엇을 해 달라는 것인가?
4. 형식: 어떤 형태로 답해야 하는가?
5. 조건: 난이도, 개수, 예제, 주석 등

## 1.8 AI가 만든 코드 이해하기

AI가 코드를 만들어 주더라도, 그 코드를 이해하지 못하면 진짜 자신의 실력이 되지 않습니다.

예를 들어 AI가 다음 코드를 만들어 주었다고 합시다.

```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name, "님!")
```

이 코드는 사용자의 이름을 입력받고 인사말을 출력합니다.

---

### 코드 한 줄씩 보기

```python
name = input("이름을 입력하세요: ")
```

사용자로부터 이름을 입력받아 `name`이라는 변수에 저장합니다.

```python
print("안녕하세요,", name, "님!")
```

저장된 이름을 사용하여 인사말을 출력합니다.

---

### AI에게 코드 설명 요청하기

다음 Prompt를 사용할 수 있습니다.

```text
아래 Python 코드를 프로그래밍을 처음 배우는 대학생도 이해할 수 있도록 한 줄씩 설명해 주세요.

코드:
name = input("이름을 입력하세요: ")
print("안녕하세요,", name, "님!")
```
