# 1.1 AI란 무엇인가?

AI는 Artificial Intelligence의 약자로
우리말로 **인공지능**이라고 합니다.

인공지능은 사람처럼

- 학습하고
- 추론하고
- 문제를 해결하도록 만든 기술입니다.

예를 들어

- 유튜브 추천
- 네이버 검색
- 번역기
- 음성비서
- 자율주행 자동차

모두 AI 기술을 활용하고 있습니다.

---

## AI는 이미 우리 주변에 있습니다.

| 서비스 | AI 활용 |
|---------|---------|
| YouTube | 영상 추천 |
| Netflix | 영화 추천 |
| ChatGPT | 질문 답변 |
| Papago | 번역 |
| Siri | 음성 인식 |

우리는 이미 매일 AI를 사용하고 있습니다.

# 🤖 AI에게 질문해 보기

ChatGPT에게 아래 질문을 입력해 봅시다.

> AI가 무엇인지 중학생도 이해할 수 있도록 설명해 주세요.

답변을 읽어보고

교재와 비교해 봅시다.

AI는 항상 같은 답을 하지 않습니다.

질문의 방식에 따라 답이 달라질 수 있습니다.

이것이 생성형 AI의 특징입니다.

---

# 1.2 생성형 AI란?

기존 AI는

사진을 분류하거나

사람을 인식하는 것이 목적이었습니다.

생성형 AI는

새로운 결과를 만들어 냅니다.

예를 들면

- 글쓰기
- 번역
- 그림 그리기
- 음악 만들기
- 프로그래밍

입니다.

우리가 사용하는 ChatGPT도 생성형 AI입니다.

---

# 생성형 AI가 할 수 있는 일

- 글쓰기

- 번역

- 발표자료 작성

- 보고서 작성

- 이메일 작성

- Python 코드 작성

- 오류 수정

- 프로그램 개선

앞으로는 대부분의 개발자가 AI를 활용하게 됩니다.

따라서 우리는

AI를 사용하는 방법을 배워야 합니다.

# 1.3 AI와 함께 코딩하는 시대

과거에는

사람이

처음부터 끝까지

직접 코드를 작성했습니다.

지금은

문제를 설명하면

AI가 초안을 작성합니다.

사람은

이를 수정하고 개선합니다.

이것이 AI 활용 코딩입니다.
