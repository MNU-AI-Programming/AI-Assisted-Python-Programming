## 📖 교재 소개

**AI 활용 코딩**은 생성형 AI(ChatGPT 등)를 활용하여 Python 프로그래밍을 배우는 입문 교재입니다.

이 교재는 프로그래밍 문법을 암기하는 것이 아니라,

- AI에게 문제를 설명하고
- AI가 작성한 코드를 이해하고
- 직접 수정하고 실행하며
- AI와 협업하여 문제를 해결하는 능력

을 기르는 것을 목표로 합니다.

모든 실습은 **Google Colab**에서 진행하며, 별도의 프로그램 설치 없이 웹 브라우저만 있으면 학습할 수 있습니다.

# 🎯 학습 목표

이 교재를 모두 학습하면 다음과 같은 능력을 갖추게 됩니다.

✅ 생성형 AI를 활용하여 Python 프로그램을 작성할 수 있다.

✅ AI가 생성한 코드를 이해하고 수정할 수 있다.

✅ Python 기초 문법을 활용하여 간단한 프로그램을 만들 수 있다.

✅ Google Colab을 활용하여 프로그램을 실행할 수 있다.

✅ GitHub를 활용하여 자신의 코드를 관리할 수 있다.

따라서

- 코드를 직접 실행해 보기
- 오류를 확인하기
- 왜 이렇게 작성되었는지 이해하기

를 반드시 수행해야 합니다.

> **AI는 여러분을 대신하여 공부하는 도구가 아니라, 함께 공부하는 학습 파트너입니다.**
