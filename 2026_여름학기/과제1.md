# 과제 1. AI와 함께 생활 계산기 만들기

## 목적

ChatGPT와 Google Colab을 활용하여 간단한 Python 계산 프로그램을 작성한다.

## 과제 내용

다음 중 하나를 선택하여 프로그램을 만든다.

1. BMI 계산기
2. 여행 경비 계산기
3. 카페 주문 금액 계산기
4. 용돈 계산기

## 필수 조건

* Google Colab에서 작성
* `input()` 사용
* 변수 사용
* 계산식 포함
* `print()` 또는 f-string으로 결과 출력
* ChatGPT에 사용한 Prompt 기록
* AI가 만든 코드에서 직접 수정한 부분 1개 이상 설명

## 제출물

1. Colab 링크
2. 사용한 Prompt
3. 최종 코드
4. 실행 결과
