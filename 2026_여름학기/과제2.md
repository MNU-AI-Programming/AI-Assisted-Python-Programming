# 과제 2. AI와 함께 좋아하는 것 목록 만들기

## 목적

ChatGPT와 Google Colab을 활용하여 Python의 `list`를 사용해 여러 개의 데이터를 저장하고 출력하는 프로그램을 작성한다.

## 과제 내용

자신이 좋아하는 것들을 입력받아 리스트에 저장하고, 마지막에 목록으로 출력하는 프로그램을 만든다.

주제는 다음 중 하나를 선택한다.

1. 좋아하는 음식 목록
2. 가고 싶은 여행지 목록
3. 보고 싶은 영화 목록
4. 읽고 싶은 책 목록
5. 이번 학기에 할 일 목록

## 필수 조건

* Google Colab에서 작성
* `list` 사용
* `append()` 사용
* `input()` 사용
* 반복문 `for` 사용
* 최소 3개 이상의 항목 입력
* ChatGPT에 사용한 Prompt 기록
* AI가 만든 코드에서 직접 수정한 부분 1개 이상 설명

## 예시 실행 결과

```text
좋아하는 음식 1: 김밥
좋아하는 음식 2: 떡볶이
좋아하는 음식 3: 라면

===== 내가 좋아하는 음식 목록 =====
1. 김밥
2. 떡볶이
3. 라면
```

## 추천 Prompt

```text
당신은 Python 프로그래밍 강사입니다.

저는 프로그래밍을 처음 배우는 대학생입니다.

Google Colab에서 실행할 수 있는 간단한 list 예제 프로그램을 만들고 싶습니다.

좋아하는 음식 3개를 input()으로 입력받고,
list에 append()로 저장한 뒤,
for 반복문을 사용하여 번호와 함께 출력하는 코드를 만들어 주세요.

각 줄에 초보자도 이해할 수 있는 주석을 달아 주세요.
```

## 제출물
